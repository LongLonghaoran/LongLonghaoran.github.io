    2.预编译:
        可以概括为： 函数声明整体提升，变量 声明提升。即：在js中任何函数和变量的声明都会被提升至代码顶端定义，对于变量，仅仅将声明提升，而函数会将整个函数定义提升。
        比如:
            这样是可以，结果会输出undefined,因为会将a的声明提升至顶端，升级为全局变量。
            console.log(a);
            var a= 10;
            这样也可以，因为会将函数的声明提升至顶端
            test();
            function test(){ console.log(123)}

        任何变量未经声明就直接赋值，都会将其绑定到window这个全局对象上成为window的属性, window就是全局的域,在全局范围内声明赋值一个变量，实际上都会将其赋值成为window的一个属性
        就是这样两种情况:
            1.在全局范围内，var a = 1;这样等价于 window.a = 1
            2.在局部范围内, function test(){ a = 1};test();这样也等价与 window.a = 1;注意在函数定义后需要调用才会执行生效

            若在全局和局部存在同名变量，优先使用局部的

            局部变量存储在AO(Activation Object)中,全局变量存在GO(Global Object)中

        window对象就是全局!
