笔记本：前端学习
1.在ES6中也规定了如何导入和导出
  1. import 模块名称 from '模块标识符'  ,  import '表示路径'
  2. 在ES6中，使用export default和export向外暴露成员

2.export default只允许向外暴露一次, 可以通过任意变量名来接收

3.通过export暴露的变量只能通过 花括号来接收
    
4.通过export可以暴露多个成员，通过花括号内使用多个 “和成员同名的变量” 以逗号分隔来接收成员
    import {title, content} from './test.js'
    
5.使用export导出的成员可以“按需接收”，可以只写 需要 接收的成员的同名名称变量
    import { title } from './test.js'

6.接收 export 导出的成员时，如果想换个名称 可以 使用 “as 别名”
    import {title as name} from './test.js'

7.结合webpack使用vue-router

8.复制一个项目的时候总是不要复制node_modules,复制完后重新npm install即可

9.webpack中使用vue-router
    1.安装
        npm i vue-router
    2.导包
        import VueRouter from 'vue-router'
    3.Vue.use(VueRouter)
    4. 绘制组件
    5.定义路由，并将组件与路由路径关联
var router = new VueRouter({
    routes: [
        {
            path: '/account',
            component: account
        },
        {
            path: '/goodslist',
            component: goodslist
        }
    ]
})
    6.将路由挂载到vm实例
var vm = new Vue({
    el: '#app',
    render: c=>c(app),
    // 将路由对象挂载到vm上
    router
})
    7.给路由组件放置占位组件
        由于这里使用的render渲染的app实例，渲染后#app会被替换为login组件,所以直接在el所控制的元素
        中写router-link是无效的
        所以应该将router-link和router-view 放置于render函数渲染的组件login中去
// app.vue
<template>
    <div>
        <h1>这是App组件</h1>
        <router-link to='/account'>账户</router-link>
        <router-link to='/goodslist'>商品列表</router-link>
        <router-view></router-view>
    </div>
</template>
    8.在/account路由下添加子路由
        1.在account之后继续添加children属性
        2.绘制组件
        3.在account组件中添加 router-link和 router-view

10.组件中style标签lang属性和scoped属性介绍
        1. scoped
            1.默认情况，在组件内写的样式是全局的样式,比如 div {color: red} ，那么就是整个页面的div都会被影响
            2.为了达到局部样式的效果，给style添加 scoped
        2.lang
            普通的style只支持普通的样式，想要启用less或sass，需要为style，设置lang
