3.包装类
    1.原始值 123'aa' 是没有属性和方法的，能调用属性和方法的一定是对象,但是除了null 和undefined。
    2.这些类型都有对象 var a = new Number(123); var str = newString('abcd'),作为对象新建之后就可以调用各类方法，比如new Number(123).toString()等等
    3.为什么var a = 123; a.len = 456;a.toString(); 这样不会报错? 在这个过程中，会隐式的new String('abc').len = 123,然后再销毁这个对象,之后再调用str.len检查的话，会是undefined,当然需要注意的是，再调用str.len的时候，仍然会执行这样一个操作 new String('abc').len,所以结果才是undefined


4.原型:
    原型是function对象的一个属性，它定义了构造函数制造出的对象的公共祖先，通过该构造函数产生的对象，可以继承该原型的属性和方法。原型也是对象。
    通过函数名.prototype来定义原型
    Person.prototype.lastName = 'Long';
    function Person(name){
        this.firstName = 'haoran';
    }
    var person = new Person();
    person.lastName就会输出公有属性。
    * 对象无法修改原型的属性，只能通过prototype进行修改

    * 通过对象获取构造函数, obj.constructor,会返回构造函数

    * __proto__属性,存放的是父类的prototype，即原型,通过对象名.__proto__可以获取原型对象

    * 可以修改对象的原型 person.__proto__ = 另一个对象

    * 可以在对象层面上手动修改其原型,修改了之后，不会影响到该原型的其它对象，比如:
        Person.prototype.name = '123';
        function Person(){};
        var person1 = new Person();
        var person2 = new Person();
        person1.name; //123
        person2.name; //123
        //修改person1.__proto__
        person1.__proto__ = {age: 18}
        person1.age  // 18
        person1.name //undefined
        person2.name // 123

    * 当然，在不手动更改对象的__proto__的情况下，所有来自于同一个原型的对象的原型都是共用的,也就是说 person1.__proto__ person2.__proto__ Person.prototype都是引用的同一个对象,可以通过 person1.__proto__ === person2.__proto__ ,===来进行研究

    * 原型的多级继承？可以像类似其他语言的继承一样，写出多级的自定义继承,比如

        Grand.prototype.name = "爷爷"
        function Grand(){
        }
        Father.prototype = new Grand();
        function Father(){
            this.lastName= '龙'
        }
        Son.prototype = new Father();
        function Son(){
            this.age= 18
        }
        var son = new Son();
        console.log(son.age);
        console.log(son.lastName);
        console.log(son.name);
        解析:
            1.new Son(),将Son.prototype 赋予 son.__proto__属性,
            son.__proto__ = Son.prototype,而Son.prototype = new Father();
            new Father() === {lastName: '龙', __proto__: new Grand()},它的prototype又为new Grand(),而new Grand()又等于:
            {__proto__: Object},其中Object.name = '爷爷'
            所以,son= new Son()最终可以等效于:
            {
                age: 18,
                __proto__:{
                    lastName: '龙',
                    __proto__: Object{
                        name: "爷爷"
                    }
                }
            }    


    * 所有对象的最终原型都是Object
