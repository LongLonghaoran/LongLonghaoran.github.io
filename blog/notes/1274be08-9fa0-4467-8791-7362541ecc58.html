笔记本：前端学习
1.container
    是bootstrap提供的一个响应式的作为版心的容器
2.container-fluid
    是一个流式布局容器,占用100%宽度
3.栅格系统
  1.     .row 行
       2.    .col-*-* 列
            第一个*表示屏幕大小
                lg 表示大屏设备
                md 表示中屏设备
                sm 表示小屏
                xs 表示任何屏幕设备下,如果前面几种都不符合的话，则会取这个,所以也可以理解为超小屏
            第二个*表示每一个列的占用的等分,每行分为12等分，所以可以取1-12中的一个
            
            bootstrap中以上几个列类型，都是采用媒体查询声明的，而且由于它采用的单条件判断的（也就是说
            每种屏幕大小的判断都只判断了最小尺寸而未判断最大尺寸），所以在只使用了一个类名的情况下，只要
            屏幕大于等于它所以适应的屏幕大小都是可以用的
                
4.栅格-----扩展
    1。栅格中的列还可以继续嵌套行列
    这是由于Boostrap的一个巧妙设计:
        container col-*-* 这些类都有 左右各15px的内边距，而row拥有-15px的外边距，也就是说将
        row放在容器container或者继续嵌套在col-*-*中之后，row会占满100%的宽度
    2.列偏移，通过 col-*-offset-* 即可让该列偏移指定等分
    3.列排序  col-*-pull-* 可将列往左偏移指定等分 col-*-push-* 可将列往偏移指定等分
    4.列偏移和列排序的区别，
        列偏移通过margin实现，所以实际上会增加元素原本占用的宽度
        列排序通过left实现，并不会增加占用的宽度，并且占用的位置不会变化
5.响应式的显示与隐藏
    visible-* :*表示屏幕大小，表示在对应屏幕大小下显示，并且在其他屏幕大小时隐藏
    hidden-*: *表示屏幕大小, 表示在对应屏幕大小时隐藏，并在其他屏幕大小时显示
    注意： 以上2个工具，二选一即可，bootstrap官方建议使用 hidden-*

6.css选择器:  +   ~
    + 紧邻的下一个兄弟元素
        div + div {}
        每一个div后面的第一个div,总体的结果就是，除了第一个div以外，其余所有的div
    ~ 后面所有的兄弟元素
        div ~ a {}
        每一个div后面的所有a标签

7.自定义字体
    
/*自定义字体图标*/
@font-face {
/*声明字体的字体名称*/
font-family: "longhr_wjs";
/*引入字体文件*/
src:
url(../fonts/MiFie-Web-Font.svg) format('svg'),
url(../fonts/MiFie-Web-Font.eot) format('eot'),
url(../fonts/MiFie-Web-Font.ttf) format('ttf'),
url(../fonts/MiFie-Web-Font.woff) format('woff');

}
/*将此类指定的元素都使用我们上面声明的字体*/
.wjs_icon {
font-family: longhr_wjs;
}
/*再使用伪元素来指定填充 "字体文件规定的与图标对应的内容"*/
.wjs_icon_phone::before {
content: "\e908";
}
.wjs_icon_tel::before {
content: "\e909";
}
