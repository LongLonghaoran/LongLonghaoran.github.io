笔记本：前端学习
1.css3
    0.现状:
        1.浏览器支持程度差，需要添加 私有前缀
        2.移动端支持优于pc端
        3.不断改进中
        4.应用相对广泛
    1.选择器
        属性选择器 伪类选择器 伪元素选择器
        1.属性选择器
            /*属性选择器：属性是相对于标签而言。所谓属性选择器就是根据指定名称的属性的值来查找元素*/
            /*1.E[attr]:查找指定的拥有attr属性的E标签。如查找拥有style属性的li标签*/

            /*2.E[attr=value]:查找拥有指定的Attr属性并且属性值为value的E标签。如想查找拥有class属性并且值为Red的li标签   =是严格匹配*/

            /*3.E[attr*=value]:查找拥有指定的attr属性并且属性值中包含(可以在任意位置)value的E标签*/

            /*4.E[attr^=value]:查找拥有指定的attr属性并且属性值以value开头的E标签*/

            /*5.E[attr$=value]:查找拥有指定的attr属性并且属性值以value开结束的E标签*/
        2.伪类选择器
            1.之前已经学习过 a:hover a:link a:visited a:active
            2.结构伪类(常用),相对于父元素或兄弟元素的伪类选择器
                相对于兄弟:
                    选择器1 + 相邻的跟着其后的第一个兄弟元素
                    .first li 表示class为first的元素后的紧跟着的第一个元素

                    选择器1 ~ 所有兄弟元素,所有在选择器1之后的兄弟元素
                相对于父元素,有两种选择方式，xxx-child xxx-of-type
                    /*相对于父元素的结构伪类*/
                    /*E:first-child:查找E元素的父级元素中的第一个E元素。在查找的时候并不会限制查找的元素的类型*/
                    /*下面这句样式查找：li的父元素中的第一个li元素
                    1.相对于当前指定元素的父元素
                    2.查找的类型必须是指定的类型*/
                    li:first-child{
                        color: red;
                    }
                    /*E:last-child:查找E元素的父元素中最后一个指定类型的子元素*/
                    li:last-child{
                        background-color: skyblue;
                    }
                    /*查找的时候限制类型  first-of-type*/
                    /*1.也是相对于父元素
                    2.在查找的时候只会查找满足类型条件的元素，过渡掉其它类型的元素*/
                    li:first-of-type{
                        color: red;
                    }
                    li:last-of-type{
                        color: orange;
                    }

                    /*指定索引位置 nth-child(从1开始的索引||关键字||表达式)*/
                    元素的的索引就是从1开始的,1表示第一个元素
                    li:nth-child(5){
                        background-color: lightblue;
                    }
                    /*偶数个元素添加背景  even:偶数  odd:奇数*/
                    /*li:nth-child(even){
                        background-color: orange;
                    }
                    li:nth-child(odd){
                        background-color: pink;
                    }*/
                    li:nth-of-type(even){
                        background-color: orange;
                    }
                    li:nth-of-type(odd){
                        background-color: pink;
                    }

                    /*想为前面的5个元素添加样式*/
                    /*n:默认取值范围为0~子元素的长度.但是当n<=0时，选取无效
                    0>>5
                    1>>4
                    ...
                    4>>1
                    5>>0*/
                    li:nth-last-of-type(-n+5){
                        font-size: 30px;
                    }
                    li:nth-of-type(-n+5){
                        font-size: 30px;
                    }

                    /*空值:没有任何的内容，连空格都没有*/
                    li:empty{
                        background-color: red;
                    }
                    li:nth-of-type(2n) {
                        background-color: #fff;
                    }

                    额外:
                        E:target 当前处于锚点处的元素会被选中
        3.伪元素:
             重点: E::before E::after
                1.是个行内元素，如果想设置宽高则需要转换成块,比如
                    display: block  转换成块级元素
                    float: position:absolute 转换成行内块
                2.必须添加content属性，即便内容为空也要添加content: ""
        4.其他伪元素:
            E:first-letter 文本的第一个字母
            E:first-line 文本的第一行
            E:selection 文本中选中的部分
        5.颜色模式之RGB
            平时设置颜色属性值时,有3种方式:
                1.预设值 red green 等等
                2.#ffffff 六位十六进制值
                3.rgb(r, g, b)
                4.HSL:
                    H 色调/色相 0表示红色 120绿色 240 蓝色
                    S 饱和度 0%-100%
                    L 亮度 0%-100% 亮度一般建议写50%
                5.RGBA HSLA 相比上面两种，增加了一个透明度参数，取值0~1
                    RGBA(255, 0, 0, 0.3)
                    它的作用在于，可以单独为元素的背景设置透明度,

                    比如opacity也是可以设置透明度，但是它是将整个元素及其
                    内部所有元素的透明度都改变

                    transparent 不可调节透明度，始终是完全透明的,它其实就是
                    代指 rgba(0,0,0,0):
                        color: transparent
        6.文本阴影
            text-shadow: offsetX offsetY blur模糊度 颜色, x方向偏移值2 y方向偏移值2 模糊度2 颜色2
            注意: 一个元素可以同时添加多组阴影效果，多组阴影会同时生效
        7.边框圆角 border-radius
            border-radius: x px x px x px x px;
            它的单位是px像素，意思是从盒子的一个角开始移动，向着盒子中心
            以水平和垂直方向各移动 x 像素，以这个点为圆心，画一个圆,也就是在一个
            xpx * xpx的正方形内画了半径为x的1/4个圆
            1/4的圆,所以当值设置宽的一半的时候，整个盒子就成为了一个正圆
                1.当只设置一个值的时候，表示4个角同时设置
                2.设置2个值,则表示分别设置了 左上+右下 右上+左下
                3.设置了3个值: 则分别表示 左上 右上+左下 右下
                4.设置了4个值: 左上 右上 右下 左下

            问题: 椭圆怎么做?
                需要分别指定圆心平移的x和y轴方向的距离
                border-radius: 100px/50px
                加一个斜杠，表示设置当前圆心不同方向的半径值
        8.边框阴影 box-shadow:
            box-shadow: h y blur spred color inset
            inset表示阴影设置为内阴影,当阴影为内阴影时,如果 h设置为负值,
            则表示内阴影从右边开始扩展
