笔记本：前端学习
1.h5新标签
    *header 页面页眉部分
    *nav 导航栏
    *footer 页脚
    *article 文章
    *section 文章中的节
    *aside 侧边栏
    *datalist 和input搭配使用,可以实现文本框的提示下拉框
        <input type="text" value="输入一下" list='star'>
        <datalist id='star'>
            <option>longhr</option>
            <option>媛媛</option>
        </datalist>
    *fieldset 将表单域内的元素打包,和legend一起搭配使用,lengend用于定义元素组的标题
    *input type新增类型: email tel number url search搜索 range 滑块
        <input type="email">
        <input type="tel">
        <input type="url" required="required">
        <input type="search">
        <input type="range" id="range"><br>
        <input type="time"><br>
        <input type="date"><br>
        <input type="datetime"><br>
        <input type="month"><br>
        <input type="week"><br>
        <input type="color"><br>
        <input type="submit"><br>
        <input type="text" placeholder="哈哈"><br>
        <input type="text" accesskey="7"><br>
    *input新增属性:
        assesskey: 设定为一个键盘上的键，比如asssekey='s',那么打开页面后只要输入alt+s即可快速定位到这个输入框
    *多媒体标签
        音频 audio
            autoplay="autoplay"自动播放 controls展示控制按钮
            loop='-1 1 2 3..'循环几次，-1表示无限次
            preload='auto'在页面就绪时就自动播放'meta'页面载入完后只载入元数据='none'页面载入后不载入音频
        视频 video
            其中可以嵌入source标签，用于浏览器遇到不兼容的视频格式时使用，会优先使用兼容的格式的视频
    * 我在百度千千音乐申请了开发者账号，申请了openapi，还在审核中
