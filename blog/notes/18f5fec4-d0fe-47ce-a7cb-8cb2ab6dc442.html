5.css3新增选择器
    1.结构(位置)伪类选择器
        :first-child在基础选择器的基础上选择首个
        :last-child最后
        :nth-child2 第2个,2可以替换为3,4,5...,但是第一个
        单词就是nth
            li:nth-child(2) {
                background-color: pink;
            }
        :nth-child(even)偶数序号的元素
        :nth-child(odd)基数序号的元素
        :nth-child(n) 选择全部的元素,表示n将从0开始
        一直到无穷大,所以就相当于从第一个元素一直到
        最后一个元素了
        同理就有:nth-child(2n),就相当于nth-child(even)
        nth-child(2n+1)就等于nth-child(odd)
        类似的用法还有:
        nth-child(4n),这样就等于选择了第4个、第8个、第12个
        。。。。这些元素了
        使用举例:
            li:nth-child(even) {
                background-color: pink;
            }
        需要注意的是 :first-child是和基础选择器贴在一起的
        ,如果之间有空格的话，那么选择是基础选择器选择到的
        元素里面的子元素
    2.属性选择器:
        [attribute]选择所有带有attribute属性的元素
        [attribute=value]选择属性等于value的元素    
        [attribute*=value]表示value在任意位置都可以
        [attribute^=value]选择以value开头作为属性的元素

    3.伪元素选择器(重点)
        如下3个了解即可，基本不用
            p:first-letter选择第一个字
            p::first-line选择第一行
            p::selection 选中状态下的样式
        after before 重点,经常用:
            p:before {
                content: ""
            }
            使用的时候必须带上content属性

            before:
                在元素"内部"的前面插入,其实这个before
                是个盒子,这个盒子是个”行内“的盒子,
                可以让它像普通元素一样给display: block;
                width: 200px;height: 200px;
            after:
                与before相对应，在盒子内容的后面放置一个
                行内盒子
            所以其实清除浮动的时候，其使用的也是“额外标签法
            ”
                .clearfix:after {
                    clear: both;
                    content: "";
                    display: block;
                    visibility: hidden;
                }
                .clearfix{
                    zoom: 1    
                }
        注意，很多网站只会写一个:，这样也可以
            p:after,因为浏览器会帮助其自动加一个:

        字体图标:
            除了之前的方法,还有一种更流行的方式:
                <span></span>
                css:
                    span {
                        font-family: 'icomoon'
                    }
                    span:after {
                        content: "图标的unicode编码"
                    }

6.css3中的盒子模型:
    box-sizing:
        以往我们添加了一个盒子div并且给其设置的宽高之后，
        如果再想给其添加padding border的话，这样势必
        会影响到其宽高，一般我们的做法就是去减少盒子的宽高
        以此来解决因为padding带来的影响    
        如果设置了box-sizing: border-box,这个时候盒子
        的padding和border对盒子本身的影响将会是“内减模式”
        也就是盒子尺寸仍然会是盒子本身设置的宽高大小
        
        box-sizing: content-box仍然和之前一样，盒子大小
        为width + border + padding

        一个特殊的场景:
            css:
                div {
                    width: 632px;
                    height: 340px;
                    box-sizing: border-box;
                }
                div:hover {
                    border: 10px solid red;
                }
                <div>
                    <img src="mi.jpg" alt="">
                </div>
            希望在鼠标悬浮时给图片周围来个边框，
            这种情况即便使用box-sizing: border-box都
            不启用，图片会被边框挤开

            这种情况应该使用伪元素来处理:
                * {
                    padding: 0;
                    margin: 0;
                }
                div {
                    width: 632px;
                    height: 340px;
                    box-sizing: border-box;
                    position: relative;
                }
                div:hover::after {
                    content: "";
                    border: 20px solid rgba(255, 255, 255, .5);
                    position: absolute;
                    top: 0;
                    left: 0;
                    width: 100%;
                    height: 100%;
                    box-sizing: border-box;
                }
