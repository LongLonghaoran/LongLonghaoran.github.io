笔记本：前端学习
1.web字体
    1.介绍
        不同浏览器所支持的字体格式是不一样的，我们有必要了解
        一下有关字体格式的知识
        首先在windows系统下，字体存储在c:/windows/fonts中

        字体格式:
            1.ttf
                windows和mac中最常见的格式
            2/otf
            3.woff
            4.eot
            5.svg
    
         网页中有一种“字体图标”的用法,之前虽然有精灵图，但是
         图片有一些缺陷：即便组成一张图但是尺寸仍然很大、
         不能很好的缩放、放大会失真、不能调色、调透明度 等等

         而如果直接使用文字的话，可以规避这些问题，优点：
         可以做出更图片一样的事情，而且可以改变透明度、旋转
         等、本质是文字，可以很随意改变颜色、阴影、透明效果
         体积更小、几乎支持所有浏览器、移动端也很强大
    2.使用    
         1.找字体
             推荐的图标网站:
                 1.iconmoon国外
                 2.阿里的iconfont,url: iconfont.cn
             可以去这些网站上下载字体。
        
         2.引用字体:
             比如去iconmoon上选中需要的图标然后下载，
             下载下来后是一个zip压缩包,解压后，fonts文件
             夹中就是需要的字体,我们只需要这个就可以了

             1. 在css中,声明字体
                @font-face {
                    font-family: 'icomoon';
                    src: url('fonts/icomoon.eot?7kkyc2');
                    src: url('fonts/icomoon.eot?7kkyc2#iefix') format('embedded-opentype'), url('fonts/icomoon.ttf?7kkyc2') format('truetype'), url('fonts/icomoon.woff?7kkyc2') format('woff'), url('fonts/icomoon.svg?7kkyc2#icomoon') format('svg');
                    font-weight: normal;
                    font-style: normal;
                }
            2.给需要字体的盒子声明font-family: 'icomoon'，这个名字是对应这font-face中
            的font-family的，名字可以自定义
            3.将demo中的小图标旁边的小盒子复制到
            小盒子里面
    3.如何追加字体图标呢？
        如果第一次下的字体文件，用到一半的时候，发现
        不够用了，该如何追加？
        答:
            继续去iconmoon网站，上传之前字体文件夹中
            的selection.json文件，这样会直接将本地那些
            图标全部展示出来，然后继续选择自己需要的
            图标，然后重新下载，再来替换原本的字体文件。

2.字体补充知识:
    字体属性可以连写,使用font属性:
        font: font-size/1.5 font-family color...
        其中font-size 和 font-family是必须写的

        font-size/1.5 1.5的意思,1.5指的是行高,表示1.5
        倍的字体大小的行高。

        有个时候一些网站初始化，会有这样的属性
        body {
            line-height: 1.5
        }
        由于行高会继承，所以页面中所有元素的行高
        都会是1.5倍行高,而不是声明具体的xx px，要不
        某些元素可能会乱

5.网站优化
    网站优化3大标签，SEO(Search Engine Optimization)缩写，意为“搜索引擎优化”，通过网站建设，代码优化等来
    提供网站的关键词排名以及公司产品的曝光度。
    三大标签:
        1.title:
            建议: 网站名-产品简单介绍
        2.description
            <meta name="description" content="ss"/>
            对于关键词的说明作用明显低些,但是由于很多搜
            索引擎仍然大量采用网页的meta标签的描述部分
            作为网站说明。
            补充在title和keywords中未提到的说明
            120字以内
            英文逗号隔开 关键词1,关键词2..
        3.keywords
            <meta name="Keywords"
            content="网上购物,网
            上商城,手机,笔记本,电脑,mp3,CD,VCD,DVD,
            存储卡,京东"/>
            一般是6-8个关键词左右
