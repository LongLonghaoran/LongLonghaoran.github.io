1.arguments.callee 函数引用自身
    在函数内部若想找到函数自身的引用，就使用arguments.callee
    例子:
        在立即执行函数中利用递归实现阶乘:
            var num = (function(n){
                if (n==1){
                    return 1;
                }else{
                    return n*arguments.callee(n-1);
                }
            })(10)

5.克隆对象，目标是将一个对象的所有自身属性拷贝到另一个对象中，且拷贝完之后，
各自的属性互不影响，即实现“深层克隆”,所谓浅层克隆就是至拷贝原始值，而像对象、数组
这类值的话，只是将引用拷贝，所以指向的对象、数组还是和原先对象是一致的。
    深层克隆:
        function clone(origin, target){
            for(prop in origin){
                if(origin.hasOwnProperty(prop)){    
                    if(typeof(origin[prop]) == "object"){
                        if(Object.prototype.toString.call(origin[prop]) == "[object Array]"){
                            target[prop] = clone(origin[prop], new Array());
                        }else{
                            target[prop] = clone(origin[prop], {});
                        }
                    }else{
                        target[prop] = origin[prop]
                    }
                }
            }
            return target;
        }

7.类数组
    有些，它实际并不是数组，但是却可以当作数组来操作，比如函数中的arguments
    对它可以通过arguments[0] 调用，可以arguments.length查看实参长度，但是，却
    不能调用push方法。
    甚至可以自己定义类数组:
        1) var obj = {
            'length': 0,
            'push': Array.prototype.push,
            'splice': Array.prototype.splice,
            'slice': Array.prototype.slice
        }
        其实，只要具备如上形式的对象，已经可以完全当作数组来使用了
    Array#push方法原理
    Array.prorotype.push = function(target){
        this[this.length] = target;
        this.length++;
    }
    面试题:
        var obj = {
            '2': 'a',
            '3': 'b',
            'length': 2,
            'push': Array.prototype.push
        }
        obj.push('c');
        obj.push('d');
        结果会是什么?
        答案: {
            '2': 'c',
            '3': 'd',
            'length': 4,
            'push': Array.prototype.push
        }
