1.关于响应类型
    responseText获取的永远是字符串形式的响应体

    response 会根据responseType的变化,xhr.responseType而变化

    responseType是在客户端，发送ajax请求之前可以自行来设置的
    比如:
        xhr.responseType = 'json'
        设置了之后，responseText responseXML就不存在了,这个时候就只剩下response了
        它的内容就是响应结果反序列化的结果

2.模板引擎案例
    1.原理
        1.如果 script标签 type='text/html'的话，那么其中的javascript代码将不会执行
        其实有一个秘密:
            html标签中的所有标签，包括像  head title meta script标签，其实我们都是通过
            设置它们的样式display: block; 再加上宽高，以此来让它们展示在页面中的

        2.所有模板引擎无外乎都一个原理，写一段模板字符串，就好比是innerHTML = `<tr><td>${a}</td><td>${a}</td><td>${a}</td></tr>`
            然后我们定义设置变量，将其作为插值插入到模板字符串中,这样我们就可以根据获得的数据a, 来动态的生成的数据了
            当然他们不一定是使用的模板字符串，但是原理差不多是这样的。

            视图模板也就是我们可以按照写html一样去正常写html，然后在其中穿插代码，在我们看来
            我们是写了一段代码一样，但是实际上模板引擎有可能都是按照字符串去读取的，然后按照它
            的规则来解析字符串

        3.为什么不写到js变量中?
            1.因为不方便，因为你将只能写js代码，不能自由地书写想要的模板代码（比如html代码)
        4.为什么要放在script标签中,而不是直接放在其他html标签中
            script标签中的内容不会显示在页面上



    2.script标签的特点:
        1.innerHTML(其中的html代码)永远不会显示在页面上,因为它的默认display属性是none

    3.使用方法：
        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <title>Document</title>
        </head>
        <body>
            <script src="template-web.js"></script>
            <script id="tmpl" type='text/html'>
                {{each comments}}
                    <tr>
                        <td>{{$value.author}}</td>
                        <td>{{$index}}</td>
                        <td>{{$value.created}}</td>
                    </tr>
                {{/each}}
            </script>
            <script>
                var xhr= new XMLHttpRequest();
                xhr.open("get", "user.php");
                xhr.send();
                xhr.onreadystatechange = function(){
                    if(this.readyState != 4) return;
                    var res = JSON.parse(this.responseText);
                    // 将res作为数据传递给模板
                    var context = {comments: res.data};
                    // 借助模板引擎api，调用方法，来渲染页面,返回的就是渲染的结果
                    var html = template("tmpl", context);
                    console.log(html);
                    // 将生成表格追加到页面的table元素中去
                    document.querySelector("table").innerHTML = html;
                }
                // 选用一个模板引擎

            </script>
            <table border='1' cellpadding="0" cellspacing="0">
            </table>
        </body>
        </html>
    4.还可以使用原始语法:
        上面使用的是标准语法，这种写法可读性更好，还有一种写法是原始语法，所谓原始语法
        就是可以在标记中直接书写Js代码
