虚函数
使用关键字virtual声明
纯虚函数
纯虚函数没有函数体，声明时使用= 0，强制派生类提供具体的实现

多态polymorphism
在基类中通过virtual定义虚函数，子类中可以重写该函数，之后通过基类对象或者指针可以调用到子类的重写方法

抽象类
如果类中至少有一个函数被声明为纯虚函数，则这个类就是抽象类。纯虚函数就是通过在声明中使用“= 0”来指定的

命名空间

函数类型、函数指针
void HelloWorld()
{
}
这个函数的类型的变量是
void(*变量名)()
看起来很奇怪，所以通常会使用auto关键来声明变量，并且直接将函数名进行赋值
auto f = HelloWorld;
或者直接使用typedef声明为一个类型
#include <iostream>

void HelloWorld()
{
    std::cout << "Hello, World!" << std::endl;
}

int main()
{
    typedef void (*function)(); // 定义一个函数指针类型
    function p = HelloWorld;
    p();
}
