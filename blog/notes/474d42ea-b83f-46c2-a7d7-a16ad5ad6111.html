helm
笔记本：Kubernetes学习笔记
kubernetes包管理器
教程:
快速入门:
1.添加一个仓库: helm repo add bitnami https://xx
2.可以搜索可以被安装的charts列表: helm search repo bitnami
3.安装chart: helm repo update && helm install bitnami/mysql --genetate-name
4.了解chart基本信息: helm show chart bitnami/mysql

使用helm:
三大概念: 
chart: helm包，包含所有部署所需的资源定义。
repo: 存放charts的地方。
release是运行在k8s中的chart的实例

helm search:
helm search hub 从artifact Hub中查找并列出charts。
helm search repo 从你添加的repo中进行查找,该命令基于本地数据搜索，无需联网

helm install:
最简单的两个参数: release名字  chart名字, 其中chart名字中需要包含repo名字，以repo-name/chart-name的形式表示

helm create:
创建自己的helm chart, helm create mychart,会创建一个mychart目录，里面含有一个空的chart具备的必要文件
helm package mychart指定chart目录进行打包，打包为mychart-0.0.1.tgz
安装helm install mychart ./mychart-0.0.1.tgz，安装本地chart

Extras:
将远端的chart下载到本地查看或者修改重新打包:
helm search hub mysql --list-repo-url 远端仓库名字,并打印远端仓库url
helm repo add https://xx  将远端仓库添加到本地
helm search repo mysql 以仓库名字搜索本地chart，这样该仓库下的chart都会搜索出来
helm pull --untardir mysql mysql/xxx 将该chart下载并解压下来到本地目录中
之后可以查看该chart内容，也可以对其修改后使用helm package来重新打包
