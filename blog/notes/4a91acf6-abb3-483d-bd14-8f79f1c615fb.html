2.2d变形/变换 transform
    translate(x,y) scale(x,y) rotate(deg)

        transform 多个效果用空格隔开

    }

5.flex伸缩布局
    1.给一个元素添加 display: flex 即表示将这个元素调整为伸缩布局的模式

6.min-width max-width
    min-width 最小宽度 min-height 最小高度
        即定义一个盒子的最小宽度，即便浏览器宽度已经小于该值，盒子尺寸仍然不会变，
        只不过超出部分看不见而已
    max-width 最大宽度 max-height 最大高度

7.背景缩放
    首先回忆一下background,它可以设置五个值 颜色 图片 平铺 位置 滚动
        background 简写属性在一个声明中设置所有的背景属性。

        可以设置如下属性：

        background-color
        background-position
        background-size
        background-repeat
        background-origin
        background-clip
        background-attachment
        background-image
        如果不设置其中的某个值，也不会出问题，比如 background:#ff0000 url('smiley.gif'); 也是允许的。

        通常建议使用这个属性，而不是分别使用单个属性，因为这个属性在较老的浏览器中能够得到更好的支持，而且需要键入的字母也更少。
    1.背景尺寸
        background-size:  length/percentage/cover/contain
        length使用 宽高两个像素值来设置,如果值设定一个值，那么第二个值会被设为auto,即为根据图片尺寸比例自动伸缩
        percentage 使用2个百分比来设置, 参照盒子的宽高,如果值设定一个值，那么第二个值会被设为auto,即为根据图片尺寸比例自动伸缩
        cover: 将背景图像一直等比缩放，扩展到足够大，以使其可以完全覆盖背景区域

8.多背景
    background可以设置多组背景属性，只需要将多个属性用 , 连起来即可
    注意： 如果多背景需要 一个共同的背景色，那么就需要 单独用一个 background-color: 来声明

9.关于行高:
    1.行高表示上下两行文本 顶线/中线/基线/底线 之间的距离（这4个距离是相等的），同时也
        表示一行文本占用的高度，因为这4条线之间永远是等距的，中文字的底部对齐基线，不会超出基线，而英文字母有一部分
        会超出基线，比如 y g
    2.行距表示两行文本的顶线与底线之间的距离
    3.在一个块级元素中，元素内的内容默认都是顶着上边框开始展示，所以当盒子高度大于
        文本高度时，文本顶着上边开始展示，文本自然也就是不是居中的了
    4.而设置行高后是这样的，行高减去字体大小之后剩余的高度值，会平均的分配到文本的上下
        两边，也就是说“在文本的行高范围之内，文本总是tr对齐的”，所以，当我们
        把行高设置的和盒子高度一致时，文本也就在这个盒子中垂直居中了
    5.但是需要注意，行高设置为和盒子高度一致后，也意为这个盒子只能显示一行文本了，
        因为这个时候一行的高度已经是和盒子高度一致了，如果文本内容超出一行了，那么
        文本开始换行，这个时候就会超出盒子范围显示了

    总结:
        文字居中，即文字内容区域的一半 + 内容区域顶部到父容器上边缘 = 父容器高度的一半。
        而内容区域顶部到父容器上边缘 = 上间距 = 下间距，所以文字内容区域 + 上间距 + 下间距 = 父容器高度。
        因为文字内容区域 + 上间距 + 下间距 = line-height，所以当line-height = height时，单行文本居中。也就是文本的中线与父容器的中间线近似重合。
