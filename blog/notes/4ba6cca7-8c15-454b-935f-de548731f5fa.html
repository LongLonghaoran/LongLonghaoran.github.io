Javascript中的
1.&& || & | 运算符
&& 与运算符,两边可以连接表达式、值都可以,需要注意的是，它返回的值总是到第一个false为止或者到最后一个值为止，例如 1 && 2 && 4 && undefined && 5,这个表达式的返回值即为 undefined,就是它总是执行到第一个判断为false的表达式
|| 或运算符，与 与运算符相反，总是到第一个true为止，或者到最后一个值为止
另外Javascript中，如下值在逻辑表达式中都表示false: undefined NaN 0 "" false

&是按位与预算运算，是将值转换为二进制然后按位进行与运算，比如  1&3 => 01 & 11结果为 01,转换为十进制即为 1,按位运算的时候，只有 11为1,其余的1&0 0&1 0&0均为 0
|是按位或运算，即将值转换为二进制后按位进行或运算，比如 2 | 4 => 010 | 100结果为 110即为6,按位或运算的时候，只有 00为0,其余的 0|1 1|0 1|1均为 1

3.循环语句
    (1)        (2)    (4)
for(var a = 1; a< 10; a++){
    (3)
    document.write(a)
}
这个循环语句的关键的注意要点在于这个语句的执行顺序:
(1) -> (2) -> (3) -> (4) -> (2) -> (3) -> (4)

所以上面的循环语句可以进行很多种形式的改写，比如下面这种
for(var a = 1; a< 10; true){
    document.write(a++)
}
for(var a = 1; a< 10000; console.log('哈')){
    document.write(a++)
    document.write("<br>")
}
其中，括号内甚至可以只写中间这一段
i = 0
for(; i < 100;){
    document.write(i++)
}

while循环，while循环其实就是只写中间部分的for循环
