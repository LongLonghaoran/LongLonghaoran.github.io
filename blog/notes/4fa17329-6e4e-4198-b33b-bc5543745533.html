0.自行完成挑错题
1.复习
    清除浮动:
        原因:
            在页面中，需要让几个盒子水平排列，
            会将这几个盒子设置为浮动的，但是
            为了不让其影响到身后的处于标准流的
            盒子（浮动后不占位置会导致之后的标准
            流盒子上移），会用一个父亲盒子包裹
            住这几个盒子，该父亲则继续用标准流，
            但是如果父亲不指定高度的话，那么父亲的
            高度将为0，这样之后的盒子仍然会
            上移造成遮挡，所以就需要清除由于
            子级盒子浮动所带来的影响。
            
        解决方案:
            1.额外标签法:
                在最后一个浮动元素之后增加一个元素
                div.clear
                并增加css:
                .clear {
                    clear: both
                }
            2. 给父亲添加overflow: hidden
            3. 伪元素法
                父元素增加class
                .clearfix:after {
                    clear: both;
                    content: "";
                    display: block;
                    visibility: hidden;
                }
                .clearfix{
                    zoom: 1    
                }

2.显示与隐藏
    1.display: none 隐藏
        将元素隐藏，并且不占用任何空间

        难点:
            给一张图片上加个蒙版!当鼠标经过时显示该蒙版
            html:
                a>div
            css:
                div{
                    w100
                    h100
                    bgc grey
                    display: none; 默认是隐藏的
                }
                关键看第二句
                a:hover div{
                    display: block;
                }
                如果希望鼠标经过图片时展示蒙版,直接
                div:hover是做不到的，因为这个时候div是隐藏
                的，根本看不到也不占空间，所以鼠标根本
                做到hover，只有其上级 a元素可以被hover,
                于是有了上面的的定义，意为a被hover时的
                div的样式

    2.visibility: visible/hidden
        对象隐藏，但保留空间

    3.overflow溢出
        盒子中内容溢出时，如何显示
        overflow: visible | auto | hidden | scroll
        visible可见
        hidden 隐藏
        scroll 添加滚轮,不过，不管内容超没超，都显示滚动条
        auto 如果没超出就不显示滚动条，超出就显示滚动条

3.cursor 鼠标样式
    cursor: default 就是正常的鼠标 | pointer
    让鼠标样式变成小手 | text | move 拖动
    键的十字架
    。。。。还有很多其他样式，pointer text move 是最常用的

4.outline轮廓线
    当文本框/域
    聚焦时，周围默认会有蓝色的线（不同浏览器不同），
    outline就是设置或取消这个线的
    outline: none取消轮廓线
    因为不同浏览器轮廓长的不一样，所以我们也可以手动更改
    轮廓，来确保不同浏览器的文本框长的差不多。

5.防止拖拽文本域

7.溢出文本显示方式

    可以继续加上
    text-overflow: clip |ellipsis
    就会添加上..省略号

    * 要实现省略号 white-space: nowrap;overflow:hidden;
    text-overflow:ellipsis;这3个属性缺一不可


8.精灵图(sprite)的本质
    当网页中图片过多的时候，网页就会加载变慢，服务器压力也会
    增大，精灵技术就是为了解决这个，就是将“许多的小图片，合
    成为1张大图片”,这样就只需要请求一次或者说尽可能少的次数了。

    比如将一个页面中所有的小图标全部放在一张大图中，然后给
    各个需要放图的地方设置它们的background: url(大图链接)
    然后再配合background-position属性，即可显示大图的不同位置
    ，以次来显示需要的图片。

9.滑动门men
    滑动门的基本结构：
        a>span a里面抱一个span
    首先准备一个两边对称的长条的图,a的背景图显示长条的左侧，span
    的背景显示长条的右侧,span的内容显示文字
