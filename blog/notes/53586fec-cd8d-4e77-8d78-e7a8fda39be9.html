关于BFC
笔记本：前端学习
1.一个利用BFC解决的问题，先不管原理
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta http-equiv="X-UA-Compatible" content="ie=edge">
  <title>Document</title>
  <style>
    .d1 {
      width: 500px;
      background-color: pink;
    }
    .d2 {
      margin-top: 30px;
      height: 100px;
    }
  </style>
</head>
<body>
  <div class="d1">
    <div class="d2">123</div>
  </div>
</body>
</html>

    
[图片/附件]
展示出来的页面有一个问题，父元素没有被因为子元素的margin-top而被撑开，这个现象也叫“外边距”塌陷

这种情况就可以通过创建bfc来解决:
    可以在父元素应用这些样式的其中之一都可以产生BFC
    overflow: hidden
    Float: left
    Border: 1px solid black
    等等


2.无副作用的BFC
    Overflow:hidden float:left这些属性加在父元素上后，除了清除浮动以外，还会附带上这个属性自带的效果
    如果只是想清除浮动而已，可以使用display: flow-root就可以做到无副作用的BFC
    
3.除了外边距塌陷，还有一种常见的需求是清除浮动，现在就很好理解之前的伪元素清除浮动法了
    
.clearfix::before, .clearfix::after {
    content: "";
    display: table;
    clear: both;
}
    然后在我们需要清除浮动的父元素上面添加clearfix作为class就可以清除浮动了
