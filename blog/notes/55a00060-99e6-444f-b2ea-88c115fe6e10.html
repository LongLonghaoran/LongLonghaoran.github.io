3、测试思路

1、绑定numa情况下的ib卡性能，及几块ib卡性能是否有差异；

2、server和client跨numa情况下的性能衰减；

3、跨主机性能与主机内的性能对比；

4、Pod和Host的性能对比


4、测试方法

server端：

ib_write_bw -ba -c UC -d mlx5_0 -i 1 --report_gbits -F -n 50000


client端：

ib_write_bw -ba -c UC -d mlx4_0 -i 1 --report_gbits -F -n 50000 10.42.39.53


gdr测试方法

可以使用perftest测试gdr，但需要使用cuda重新编译perftest。

git clone https://github.com/linux-rdma/perftest.git
 
cd perftest/
./autogen.sh && ./configure CUDA_H_PATH=/usr/local/cuda/include/cuda.h && make -j
