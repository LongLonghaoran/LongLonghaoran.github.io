1.Jquery特殊属性操作
    1.val方法
        操纵value属性的方法
        一般value的属性不会使用attr("value", xx)去操作,因为比较麻烦
    2.html text
        html相当于 innerHTML
        text相当于 innerText
    3.重点 Jquery中的三大家族 之后可以来复习下 Jquery中的css("width") width() offset().width() scollLeft() clientX()方法
        1.width 与 height 家族
            获取内容区的宽度
            1.设置宽度 width(200)
            2.获取宽度 width()
        4.innerwidth
            获取padding + content的width值
        5.outerwidth
            获取 border +padding + content的值
            如果还要继续包含margin，可以传个参数true即可
            outerwidth(true)
        6.offset()方法
            效果 类似 js的offset,它会返回一个对象 {top: 8, left: 8}
            其中包含了left 和top值,但是它不会相对于有定位的父盒子的left top值，
            它总是返回相对于页面边界的top left值
        7.position()
            效果同 js的offset,获取相对于有定位的父盒子的left top值,形成一个对象
        4.scrollXX() 家族
            设置或获取滚动条的位置
            $(window).scrollTop()可以获取、设置整个页面的滚动距离
            通过animate来设置页面滚动的动画时，无法通过
                $(window).animate({scrollTop: 0})来设置，只能通过
                $("html,body").animate({scrollTop:0})来设置，因为
                window的dom对象并没有一个scrollTop属性,而body 和 html元素
                是有的,之前获取页面scroll的时候，就是通过
                document.documentElement.scrollTop
                document.body.scrollTop
        5.clientXX() 家族
    4.Jquery事件机制
        1.以click为例，有几种方式
            简单 事件
            1.click(function(){})
                缺点，一次只能注册一个事件，而且对于通过Jquery新增的元素无法为其自动注册事件
            2.bind("click", function(){}) 弃用（旧版Jquery方式)

            -------上面注册事件都是简单事件，有个缺点 对于通过Jquery新增的元素无法为其自动注册事件

            3.delegate事件委托
                它可以解决上面提出的问题
                给需要注册事件的元素的父元素注册一个委托事件
                $(父元素).delegate(目标元素选择器, "click", function(){});
                这样子即便新增了新的元素也仍然能够自动的注册已有的事件了
                它的原理是通过事件冒泡做到的
                优点:
                    1.相比起简单事件，它只需要给父元素注册一个事件即可，而简单事件实际上是
                        给所有Jq对象各注册了一个事件
                    2.可以动态地给新增的元素注册事件
                缺点:
                    1.只能注册委托事件
            4.重点 on方法 最为推荐的方法，on方法综合了所有事件注册的方式
                on("click", function(){}) 最为推荐的方式
                优点: 一次能够注册多个事件，可以注册委托事件，写法简单

                1.简单事件
                    $().on(事件类型名称, fn)
                    给自己注册一个事件
                2.委托事件
                    $().on(事件类型名称, 选择器, fn)
                    给自己的子元素注册事件

        2.事件的执行顺序
            元素自身的简单事件 -》 委托事件事件 -》父元素自身被冒泡的同名事件

        3.重点 事件解绑
            $().off() 方法
            也可以传一个参数，表示要移除的事件类型名称

        4.事件触发
            1.直接调用 click() hover()等方法
            2.$().trigger(事件名称) 推荐;

        5.事件对象
            Jq事件回调函数和js一样，也可以给方法设置一个形参来接受事件对象
            $().on("click", function(e){

            });
            1.event.data 很少用
                可以通过事件注册方法传入data参数
                $().on("click", {a:1}function(e){
                    console.log(e.data.a);
                });
            常用 2.阻止冒泡行为
                e.stopPropagation()
            常用 3.阻止默认行为
                e.preventDefault()
            常用 4.既阻止冒泡又阻止默认行为
                return false;
        6.end方法
            获取Jquery对象的前一个状态，和前一个jq对象
            因为Jquery中可以链式调用，但是当我们调用prev() prevAll()  nextAll() 这些方法时
            这个时候的返回值已经不是最初调用方法的那个jq对象了，所以可以通过end()来继续获取
            当然，最好的方法是我们重新来一行代码，再次从最初的那个jq对象开始调用

        7.each方法
            遍历jq对象的方式,可以直接调用each方法
            $().each(function(index, ele){
                //index是元素在所有兄弟元素中的索引/序号
                //ele下标对应的dom对象,注意是dom对象,如有需要可以使用$(ele)转为jq对象
            })
        8.$冲突的解决方法
            jq中, $ 已经被定义为了一个方法, 其实在一些别的框架中有可能也定义了$这个函数，
            就是名称有可能冲突
            首先将Jquery放在最后引入
            1.jq中可以调用$.noConflict() 来释放控制权，就会将$这个名称让出给其他js文件
            2.可以使用 Jquery来继续调用 Jquery("div")
            3. var $$ = $.noConflict() noConflict()方法有一个返回值，返回的是新的Jquery函数的别名
               这时我们可以使用$$来替代 $,用法是完全一样的

           4.前提条件是Jquery一定要放在最后引入
