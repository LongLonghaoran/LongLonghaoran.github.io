js基础
1.js代码怎么写
    1.可以分3个地方写:
        1.页面script 标签中
        2.普通标签的onclick等事件属性中,这叫做 内嵌代码
        3.额外的js文件中，写好后在页面中引入进来
    2.在同一对script标签中有错误代码的话，那么该错误代码后面的js将不会执行
    3.一对script中的js代码执行错误，并不会影响后面的其他script标签中的js代码
    4.写script标签的时候，最好是加上
        type='text/javascript' language='JavaScript'这两个属性,以防浏览器不兼容
    5.script标签在页面中可以出现多对
    6.script一般是放在body中最后，有时会在head标签中,因为页面是从上到下执行的，放在body
        之后，可以确保页面元素都加载完毕了
        <body>
            页面内容。。。
            <script></script>
        </body>
        当然如果将script标签写在body 或者html之后的话，浏览器也会自动将其移到body中的最后
        位置
    7.如果script标签是引入外部js文件的作用的话，那么这对标签中就不要写任何的js代码，
        如果要写，就重新写一对script标签

2.变量

    2.js中声明变量都用var,使用var 声明后就在内存中开辟一个存储空间
    3.var可以一次性声明多个变量
        var x,y,z,k,j
    4.js中区分大小写,long lonG 是两个不同的变量
    5.变量名有一定的规范:
        1.以字母 $ _ 开头
        2.中间或者后面可以有 $ 字母 数字
        3.变量名采用 驼峰式 命名
    6.声明多个变量并且赋值
        var x=1,y=2,z=3
3.在webstorm中打开html时，会自动启动一个 小型的服务

4.交换两个变量:
    var a = 1,b = 2;
    1.用一个临时变量 var temp; temp =a; a=b;b=temp;
    2.对于数字的交换还可以使用这种方式,不借助第三个变量
        a = a+b
        b = a-b
        a = a-b
    3.使用位运算
        使用位运算符^,异或运算符
        a = a^b / a^=b
        b = a^b / b^=a
        a = a^b / a^=b
        上面的代码也可以简写为一行 a = (b = (a ^= b) ^ b) ^ a;
        原理:
            var a = 1, // 二进制：0001
             b = 2; // 二进制：0010
            a = a ^ b; // 计算结果：a = 0011, b = 0010
            b = a ^ b; // 计算结果：a = 0011, b = 0001
            a = a ^ b; // 计算结果：a = 0010, b = 0001
            本题巧用位运算的技巧，利用 a ^ b ^ b == a 的特点，进行数值交换，避免了使用算术运算带来的弊端，不会发生溢出问题。
5.webstorm格式化代码快捷键: Ctrl + Alt + L, 不过，前提是代码格式是正确的

6.js的原始数据类型,总共有6种
    number 数字类型(整数和小数)
    string 字符串类型
    boolean 布尔类型
    null 空类型,值只有一个: null
    undefined 未定义，值只有一个
        什么情况下会有undefined
            1.变量声明了，没有赋值
            2.函数没有明确返回值，如果接受了，结果也是undefined
    object 对象
    但是唯独null有点特殊,typeof null == 'object',null表示不指向任何值,也可以说是
    表示一个不指向任何值的 对象

7.进制
    js中可以表示哪些进制：
    1.10进制
        正常写十进制即可
    2.八进制
        0开头, var num = 010; //输出结果是十进制的8
    3.十六进制
        0x开头, 如 var num = 0x10; //输出结果是十进制的16

8.number类型
    1.数字类型有范围:
        Number.MAX_VALUE 获取最大值
        Number.MIN_VALUE 获取最小值
    2.精度问题,在js中不要用小数去验证小数:
        一个例子:
            0.1+0.2 != 0.3
        结果并不等于0.3
    3.NaN并不会等于NaN, NaN == NaN 结果将会是false,可以这样理解,NaN表示的是
        所以，不要用NaN是不是NaN,如果有需要应该使用: isNaN()
    4.isNaN(x) 方法，用来判断一个变量 是不是一个NaN(非数字),只要x代表的的确不是一个
        数字，那么结果就为true,像  "s" "s1" undefined NaN 这些都是非数字 NaN
        注意 "10"这样的字符串会有自动类型转换，也算是数字

9.字符串拼接
    字符串可以通过+进行拼接，只要+两边只要有一个字符串就会进行字符串拼接
    注意: 只有 + 才叫拼接，其他的数学运算符比如 - * /这些的话，会按照隐式转换
    的规则来进行

10.关于null值，唯一一种可以获取null的方式就是手动赋值为null,如 var a = null;


11.类型转换
    1.其他类型转数字:
        1.parseInt(x) 转为整数
            如果x是字符串的话，那么只要第一位是个数字的话，就可以转为数字,
            结果是从第一位开始的最大的整数
        2.parseFloat(x) 转小数
            与parseInt一样，不过可以转换到小数位
        3.Number 转数字,这个方法的转换规则就和一般的编程语言一致,转换的字符串
            必须是完完全全的数字才行 Number('1') // 1
    2.其他类型转字符串
        1. num.toString()
            只有有意义的值才可调用,比如 var num =1;num.toString();
            像 null undefined就不可以调用
        2. String(x)
            任何值都可以转换
    3.其他类型转换为布尔类型
        Boolean(x)
            0 NaN null undefined "" 这五种会转换为false,其他的都转换为true
            非数字 空 未定义 空字符串
