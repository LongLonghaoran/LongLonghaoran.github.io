Tensor Core出现背景

tensor是一个数学概念。0维的tensor就是标量；1维的tensor是向量；2维的tensor是矩阵。

[图片/附件]

[图片/附件]

AI中经常需要做的就是矩阵计算，特别是LLM模型。

[图片/附件]

[图片/附件]

也就是一个简单的的矩阵的乘法需要64个乘法以及48个加法运算。如果是这种模式，还需要额外的16个加法运算。

传统的做法是利用Cuda来并行化计算，达到加速的目的。最早的GPU只有渲染功能，想要做计算只能通过图形API来驱动，被称为GPGPU时代。

后来，NVIDIA率先开放了GPU的通用计算能力，能利用CUDA编程语言来驱动。这时候GPU的核都是Cuda Core。由于一个GPU里面有大量的cuda core，使得并行度高的程序获得了极大的并行加速。但是，cuda core在一个时钟周期只能完成一个操作，类似上面的矩阵乘法操作依然需要耗费大量的时间。

为了进一步加速AI中大量的GEMM(general matrix multiplication)，NVIDIA在2017年推出了Volta架构的GPU，因为这个架构的GPU瞄准的是专业领域，完全没有使用家用的GeForce架构。也是在这个架构里Tensor Core被引入。它可以在一个时钟周期完成两个半精度浮点矩阵的乘法(64 GEMM per clock)。

[图片/附件]

Volta GV100 Streaming Multiprocessor(SM)

Streaming Multiprocessor(SM):

  • 64 FP32 + INT 32 cores
  • 32 FP64 cores
  • 8 tensor cores (fp16/fp32 mixed-precision)

4 partitions inside SM

  • 16 FP32 + INT32 cores each
  • 8 FP64 cores each
  • 8 LD/ST units each
  • 2 tensor cores each
  • each has: wrap scheduler, dispatch unit, register file

Volta的小升级版本是Turing架构，它跟Volta非常类似。Tensor Core可以在一个时钟周期完成64个半精度浮点的乘加操作。

[图片/附件]

Turing TU102/TU104/TU106 Streaming Multiprocessor (SM)

Streaming Multiprocessor(SM):

  • 64 FP32 + INT32 cores
  • 8 Turing tensor cores (FP16/32, INT4/8 mixed-precision)
  • 1 RT (ray tracing) core

4 partitions inside SM

  • 16 FP32 + INT32 cores
  • 4 LD/ST units each
  • 2 Turing tensor cores each
  • each has: warp scheduler, dispatch unit, 16K register file

往前迈进一大步的是Ampere架构，用得比较多的A100就是这种架构。它可以支持一个时钟周期完成两个矩阵的乘法操作。

[图片/附件]

GA100 Streaming Multiprocessor

GA100 Streaming Multiprocessor(SM):

  • 64 FP32 + 64 INT32 cores
  • 32 FP64 cores
  • 4   generation tensor cores

4 partitions inside SM

  • 16 FP32 + 16 INT32 cores
  • 8 FP64 cores
  • 8 LD/ST units each
  • 1 generation tensor core
  • each has: warp scheduler, dispatch unit, 16K register file

Ampere架构有很多类型的GPU，GA100， GA102， GA103， GA104， GA106， GA107， GA107S，GA107SB， GA10B。上图的GA100就没有rt core，但是GA102就有。详情可以参考techpowerup.com。

Tensor Cores

Tensor Core是可编程的混合精度矩阵乘加运算单元。

Mixed-precision, fast matrix-matrix multiply and accumulate (mma)。

例如Volta支持大小的矩阵的混合精度乘加: ，其中矩阵A和B都是fp16精度，矩阵C和矩阵D是fp16或者fp32精度。

[图片/附件]

每个tensor core在单个时钟周期进行64个浮点fma操作。

[图片/附件]

在编程的时候，可以使用tensor core API。

Low-level选择为：

  • CUDA C WMMA （warp-level matrix multiply and accumulate)
  • PTX WMMA and mma (needed for some features) instructions
  • SASS hmma instructions (not documented)

High-level选择为

  • NVIDIA CUTLASS (template abstrctions for high performance matrix-multiplies)
  • NVIDIA cuBLAS
  • NVIDIA cuDNN
  • Integration into TensorFlow
