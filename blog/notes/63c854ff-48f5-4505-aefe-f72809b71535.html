Elasticsearch入门之:聚合
笔记本：ElasticSearch
聚合搜索在request body中使用aggs来执行,
一个基本的格式是:
    {
        “aggs”:{
            “聚合的名字”: {
                “聚合的类型”: {
                    “field”: “字段名”
                }
            }
    }
[图片/附件]
以上就是一个基本的分组,会按指定的字段进行分组，会返回指定字段的值以及对应的数量

关于什么叫Terms聚合？

terms聚合，它是按照某个字段中的值来分类

比如性别有男、女，就会创建两个桶，分别存放男女的信息。默认会搜集doc_count的信息，即记录有多少男生，有多少女生，然后返回给客户端，这样就完成了一个terms得统计。

聚合又分为两种聚合，对应到RDBMS的group by和相关的聚合函数
