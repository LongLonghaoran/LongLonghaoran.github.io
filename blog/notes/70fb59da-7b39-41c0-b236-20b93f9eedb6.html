1.作用域
    函数其实也是一个对象，它有一些属性,比如length(获取形参长度),name(获取函数名称)
    也有一些隐藏属性,比如test.[[scope]]
    [[scope]]:每个javascript函数都是一个对象，对象中有些属性我们可以访问但有些不可以,这些属性仅供javascript引擎存取,scope就是其中一个。scope指的就是我们所说的作用域,其中存储了运行期的上下文的集合。

    作用域链：[[scope]]中所存储的是执行期的上下文对象的集合，这个集合呈链式链接，我们把这种链式链接叫做作用域链。

    系统每调用一次函数就会生成一个执行期上下文，每次调用的执行期上下文都是独一无二的，所以多次调用一个函数会导致创建多个执行期上下文.当函数执行完毕，它所产生的执行期上下文被销毁。

    变量调用时，会在当前函数的作用域链中依次向下查找，函数自身的执行期上下文AO排在作用域链中的第一位,然后时GO全局作用域，如果函数内还有一个函数定义的话，那么
    这个函数在定义的时候，会首先生成一个自己的AO,其作用域链继承其父函数的作用域即  AO -> GO,然后在此基础上再生成自己的作用域AO放在作用域链的最顶端。

    一个复杂的例子:
        function (a){
            function b(){
                var bbb = 234;
                console.log(aaa);
            }
            var aaa =123;
            return b;
        }

        var glob = 100;
        var demo = a();
        demo();

2.立即执行函数
    只被执行一次javascript，针对初始化功能的函数。
    格式:
        (function(){

        }())  //要注意在{}后面还有一对()

    这样定义的函数只会执行一次，在浏览器的体现为，在(function(){}())中定义的变量，在浏览器控制台中输出会报错undefined,而不写在(function(){}())中的变量时可以找到的。
    * (function(a, b, c){}(1, 2, 3))立即执行函数也可以带参数。
    * 可以有返回值，var num = (function(a, b, c){ return a+b+c}(1, 2, 3))
    *立即执行函数的第二种写法:
        (function(){})()
    理论上与第一种方式时等价的,第二种方式表示，在函数定义后通过执行符号()将函数立即执行。
    * 只有表达式才能被执行符号执行
    * var a = function(){}();这样也会执行，但是函数表达式如果执行符号执行后
    那么它的名称将会被忽略,即a为undefined报错
