安装并配置agent客户端
笔记本：Puppet
安装agent:
    在*nix上安装:
        sudo yum install puppet-agent
    启动puppet service:
        sudo /opt/puppetlabs/bin/puppet resource service puppet ensure=running enable=true

配置agent:
    1.配置PATH来访问puppet 命令
        puppet的cli由单个puppet命令以及许多自命令组成，例如puppet —help
        puppet命令位于 /opt/puppetlabs/bin/下
    2.配置server设置
        server是唯一需要的设置项，它允许agent连接到server端
        设置方法：
            命令: puppet config set server puppetserver.example.com —section main
            手动设置:
            /etc/puppetlabs/puppet/puppet.conf
            在[main]区块下设置
            sever= puppetserver.example.com
            注意将主机域名替换为你实际的主机域名

    3.将agent连接到主服务并签发证书
        一旦添加好server配置后，你必须将agent连接到主服务端以定期检查报告状态，获取它的编目文档，以及更新它所需要的配置
        3.1 连接
                puppet ssl bootstrap
        3.2 在主服务的签发证书
                sudo puppetserver ca sign —certname <name>
        3.3 在agent节点，再次运行agent
            puppet ssl bootstrap
