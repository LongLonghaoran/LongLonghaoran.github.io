apipie
笔记本：rubygems

什么是apipie?
apipie是一个用于给restful api做文档的dsl和rails引擎.不同于传统的在代码中写注释,apipie让你描述代码,贯穿在代码中.这会带来一些有点比如:
1.无需学习新的语法结构
2.复用文档用于其他目的
3.更容易扩展和维护
4.复用其他资源来用于文档（比如路由 等）
文档还可以在web页面上访问

常见方法

  1. param方法:用于定义方法参数
  2. def_param_group 用于定义需要复用的方法参数
  3. param_group 用于使用需要复用的方法参数
  4. api 定义 api 路径
  5. returns 定义响应内容和标头等
  6. resource_description 在控制器层面定义api详情,是所有api方法的gong

注意事项

apipie不仅仅可以生成api文档,还可以用于真的来校验api调用的参数
比如 param :name, String 用于校验name字段必须为字符串类型,在调用接口时如果类型不为字符串的话,那么是会抛出paramInvalid的异常的
