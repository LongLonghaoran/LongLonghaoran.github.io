1.switch case语句
    switch(n){
        case x:
            语句1;
            break;
        case y:
            语句2;
            break;
        case z:
            语句3;
            break;

    }
    用于判断n是否等于下面case的值，需要注意的是，除了最后一个case以外，每个case后面都必须跟个break,不然在执行符合条件的case语句之后，之后的case语句仍然会继续执行。

3.对象，Javascript很重要的一个角色
    var obj = {}，对象是用来存放数据，格式是花括号内部包含着 key: value,
    key可以是字符串，也可以是直接的字符。
    格式：
        {
            属性名 冒号 属性值 英文逗号    
        }
    总之就是使用key-value的形式的来存储各类数据，格式中需要注意的点就是首先属性名称可以是包含也可以不包含''或""，然后后面紧跟 :,接着是属性值value,如果后面还要继续写key-value的话，那么一定要跟个, ,而不能是;这些。


4.编程形式的区别：
    1.面向过程的编程
        做事情的时候首先分析做一件事情的步骤，然后去编程按步骤来实现达到目标，比如：
        建一栋房子，我会想到首先打地基，构造材料、雇佣人员，搭框架，灌泥沙。。。然后基于这个相反，我会去写出这样的程序：
        function 建房子(){
            打地基;
            构造材料;
            雇佣人员;
            搭框架;
            灌泥沙;
            ....
        }
        这样就是典型的面向过程的编程。
    2.面向对象的编程
        面向对象就做事情的时候，将事情分成不同的，先单独去想各部分的事情该怎么实现，然后最后再来统筹使用即可，上面的建房子示例，按照面向对象的思想去写的话，大概会写成这个样子：
        function 打地基(){

        }
        function 构造材料(){

        }
        function 雇佣人员(){

        }
        function 灌泥沙(){

        }
        function 建房子(){
            打地基
            构造材料
            雇佣人员
            ...
        }

6.类型转换
    显示类型转换,有几个方法，用于转换数据类型
        1.Number方法
            *将方法的参数转换为数值，不能转换的统一转换为NaN，
            *需要注意的是，Number是非常重要的一个转换方法，很多时候隐式转换和一些显示其他方法内部都是使用的Number先进行转换的，比如 isNaN(n), 其内部是先使用Number将n进行转换，然后再判断其是否是NaN;true转为1,false转为0,null转为0, undefined转为 NaN，
            * 会转为NaN的情况: 以非数字打头的字符串，undefined, 字符串中含有非数字字符

        2.parseInt 
                * 将值转换为整数,对于小数会直接取整数部分不会四舍五入
                * parseInt(value, radix),radix表示基地，可选参数，即radix进制，比如parseInt(10,16)，则表示将value看作是16进制数，然后将其转换为10进制的整数,ra
                * dix的取值范围是2-36
                * parseInt转换规则是从value的数字位开始一直查到非数字位为止,即如果一个字符串前面几位都是数字，而后面不是数字，比如 “123a&”也是可以转换的
        3.parseFloat 同parseInt，转换的结果是Float
        4.toString 该方法的调用者是被转换的变量本身，而不是将其作为参数传入，需要注意
        的是，必须将需要转换的原始值赋值给变量之后，由变量来发送方法，而不能直接在原始
        值上面调用toString(),否则会报错，比如： 1.toString(),必须先将其赋值给变量再调用: a = 1; a.toString();
        b=true;b.toString();另外还有一点需要注意的是:
        undefined和null即便赋值给变量也无法调用toString()，否则会报错
        toString(radix),该方法也有一个可选参数，radix表示一个整数值，如果被转换的数
        也是一个数值的话，会将被转换数当作十进制数值先转换成目标进制再转换成字符串，比
        如: a = 10; a.toString(4)
        利用parseInt(n, radix)和toString(radix)可以将任何进制的数转换成任何进制的数:
        var a = n进制的数;
        var m = parseInt(n, n).toString(m);即为将n进制转换为m进制的数。
    隐式转换:
        ++/-- -（减) +(正)/-(负),但是！需要注意只有+加号特殊，+在碰到字符串时会将两边都转为字符串进行处理
        +加号，只要两侧有一个是字符串就转为字符串
        -*/% -> number,使用Number进行类型转换再进行运算
        && || !
        == ，使用==以及上面的二元运算符的时候，当运算符两边连接的是类型不一致的值时，都会先进行隐式类型转换然后再进行运算,总体来说，尤其需要的注意的就是，当一个数值和一个非数值进行运算的时候，将非数值使用Number进行转换，然后再进行运算，但是 + 除外，只要有一个时字符串就转换为字符串

        === 三等号，这样的比较表达式，两边的变量均不会进行任何的类型转化，比如 1 === "1"是false,而 1 == "1" 1 == true结果是 true
