笔记本：前端学习
1.网络接口:
    比较简单，就是本机网络连接或者断开时会触发的事件，有online offline两个事件
    使用方法:
            <script>
                window.addEventListener("online", function () {
                    alert("网络连通了");
                });
                window.addEventListener("offline", function () {
                    alert("网络断开了");
                });
            </script>
2.全屏接口:
    方法:
        requestFullscreen():开启全屏显示
        cancelFullscreen():关闭全屏状态
        fullScreenElement:是否是全屏状态
    但是，由于不同浏览器对它的支持程度不一样，所以在实际使用时需要加上前缀
        chrome:webkit firefox:moz ie:ms
        比如:
            webkitRequestFullcreen()
            mozRequestFullScreen()
            msRequestFullscreen()
        如果需要同时支持多款浏览器，则需要使用js进行判断:
            使用能力测试添加不同浏览器的前缀:
                div = document.querySelector("div");
                if (div.requestFullScreen){
                    div.requestFullScreen();
                }else if(div.webkitRequestFullscreen){
                    div.webkitRequestFullscreen();
                }else if(div.msRequestFullscreen){
                    div.msRequestFullscreen();
                }
            退出全屏显示的话，必须使用document来退出，判断是否是全屏状态
            也是只能通过document.cancelFullScreen();来操作
3.Filereader
    读取文件内容
        1.readAsText()读取文本文件,返回文本字符串，默认编码是utf-8
        2.readAsBinaryString():读取任意类型的文件,返回二进制字符串,一般用来存储
        文件，而不是给用户看，因为读取出来的都是二进制数
        3.readAsDataURL(): 读取文件获取一段以data开头的字符串，这段字符串的本质
        就是dataURL是一种将文件嵌入到文档的方案.DataURL将资源转换位base64编码的
        字符串形式，并且将这些内容直接存储在url中
            1.该方法没有返回值,它会将读取的结果存储在文件读取对象的result属性中
            2.该方法需要一个参数Blob,什么叫Blob呢，它是binary large object,表示
            一个二进制的大文件，说白了就是文件
            3.文件存储在file表单元素的files属性中
            4.读取文件是有读取时间的,所以需要配合相关的事件来获取文件数据
                onload:读取成功时触发
                onloadend:读取完成时触发，无论成功或失败
                onloadstart:开始读取时触发
                onabort:读取中断时触发
                onprogress:读取文件过程中触发
        4.abort():中断读取
    应用场景:即时预览
        当用户上传一张图片后，就立刻进行展示
            <form action="">
                文件:
                <input type="file" name="myFile" id="myFile" onchange="getFileContent();"><br>
                <img src="" style="width: 200px;height: auto;" alt="">
                <input type="submit">
            </form>
            <script>
                function getFileContent() {
            //        创建文件读取对象
                    reader = new FileReader();
                    reader.readAsDataURL(document.querySelector("#myFile").files[0]);
            //        读取文件，获取DataURL
            //        监听reader是否已经读取
                    reader.onload = function () {
                        console.log(reader.result);
            //            展示图片
                        document.querySelector("img").src = reader.result;
                    }
                }
            </script>
        核心概念:
            1.即时性通过oninput实现
            2.展示通过readAsDataURL实现
4.拖拽:
    1.除了图片和超链接默认可以拖拽以外，其他的元素增加draggable='true'即可拖拽
    2.拖拽的实质是通过拖拽事件来实现的
        在拖放的过程中会触发以下事件：
        a)在拖动目标上触发事件 (源元素):
        ondragstart - 用户开始拖动元素时触发
        ondrag - 元素正在拖动时触发
        ondragend - 用户完成元素拖动后触发
        b)应用在目标元素上的事件—   当拖拽元素在目标容器上进行操作的时候:
        ondragenter - 当被鼠标拖动的对象进入其容器范围内时触发此事件
        ondragover - 当某被拖动的对象在另一对象容器范围内拖动时触发此事件
        ondragleave - 当被鼠标拖动的对象离开其容器范围内时触发此事件
        浏览器默认会组织ondrop,我们必须在ondragover中阻止浏览器的默认行为
        ondrop - 在一个拖动过程中，释放鼠标键时触发此事件
            简单实例:
                    <div class="div1" id="div1">
                        <p id="pe" draggable="true">试着把我拖过去</p>
                    </div>
                    <div class="div2" id="div2"></div>
                    <script>
                        var p = document.querySelector("#pe");
                        var div2 = document.querySelector("#div2");
                        var div1 = document.querySelector("#div1");
                        p.ondragstart = function () {
                            console.log("ondragstart");
                        };
                        p.ondragend = function () {
                //            console.log("ondragend");
                        };
                        p.ondragleave = function () {
                            console.log("ondragleave");
                        };
                        p.ondrag = function () {
                //            console.log("ondrag");
                        };
                        div2.ondragenter=function(){console.log("ondragenter")};
                        div2.ondragover=function(e){e.preventDefault();};
                        div1.ondragover=function(e){e.preventDefault();};
                        div2.ondragleave=function(){console.log("ondragleave")};
                        div2.ondrop=function(){
                            div2.appendChild(p);

                        };
                        div1.ondrop=function(){
                            div1.appendChild(p);

                        };
                    </script>
            扩展,实现任意个元素可以自由拖动到任意个框中:
                通过document.onXXX即可，对document绑定这个事件
                然后再配合e.target获取事件源即可
                1.给方法增加参数e,然后使用e.target即可
