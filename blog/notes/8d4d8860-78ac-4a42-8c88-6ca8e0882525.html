    debian 8,9,10
    ubuntu 16.04
    sles 12 sp1
注意：不同于puppet agent,puppet server不支持macOS

安装puppet server
    puppet server被配置为使用2gb内存。如果你只是在虚拟机中测试一下安装流程而已，这么多内存是没必要的。要更改内存分配额度的话，请看Running Puppet Server on a VM.
    
安装:
    yum install puppetserver
开启服务:
    systemctl start puppetserver
查看版本:
    puppetserver -v
