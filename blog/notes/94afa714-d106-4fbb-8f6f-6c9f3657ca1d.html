Kubevirt-Quickstart
笔记本：Kubevirt
在k8s环境中安装kubevirt:
# Point at latest release
$ export RELEASE=$(curl https://storage.googleapis.com/kubevirt-prow/release/kubevirt/kubevirt/stable.txt) 
# Deploy the KubeVirt operator
$ kubectl apply -f https://github.com/kubevirt/kubevirt/releases/download/${RELEASE}/kubevirt-operator.yaml 
# Create the KubeVirt CR (instance deployment request) which triggers the actual installation
$ kubectl apply -f https://github.com/kubevirt/kubevirt/releases/download/${RELEASE}/kubevirt-cr.yaml 
# wait until all KubeVirt components are up
$ kubectl -n kubevirt wait kv kubevirt --for condition=Available

如果硬件虚拟化不可用(比如在虚拟机内部,cat /proc/cpuinfo中没有显示vmx等?),则会启用软件模拟备选方案,通过kubevirt对象的spec.configuration.developerConfiguration.useEmulation设置为true即可
从源码部署:
自己看吧: https://github.com/kubevirt/kubevirt/blob/main/docs/getting-started.md,下载kubevirt源码，构建镜像然后放到k8s中运行
