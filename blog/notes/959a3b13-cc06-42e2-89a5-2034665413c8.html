函数:
    高内聚 弱耦合
 

定义方式:
    function test(){}
    1.命名函数表达式
    var test = function abc(){}
    2.匿名函数表达式
    var test = function (){}

定义函数时的参数个数与调用函数的参数个数可以不必相等，比如:
1.
function sum(a, b, c, d){}
sum(1)
这样不会报错，只不过在函数内部 b c d都将为undefined
2.
    function sum(a){}
    sum(1,2,3,4)
这样也不会报错,在函数内部可以通过arguments来获取所有的参数,arguments是个数组

3.length方法，函数有个length方法，通过函数名进行调用，可以获取形参的个数，而arguments.length获取的实参的个数

4.arguments和形参是同时变化的，即通过改变 arguments或者形参，另外一个同样的参数也会变化。

function sum(a, b){
    arguments[0] = 10;
    console.log(a)
}
sum(1)
# 输出10
但是只有当形参被实参赋值上的时候，形参和arguments才会有这种对应关系

5.全局变量，局部变量：
    全局变量： 定义在script顶部的变量
    局部变量：定义在函数内部的变量

    在函数内部可以访问到在函数外部定义的变量

    输入一个10位以下的数字，输出它的中文读法，如 10000就输出 壹万元
