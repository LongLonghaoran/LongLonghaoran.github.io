2.全局组件定义的三种方式:
    1.Vue.extend
        1.Vue.extend({template: 'html内容'})创建模板
        2.Vue.component('myCom1', com1)
        3.使用
            直接将组件名称作为标签名使用，需要注意：
            组件名称如果是驼峰式命名，那么作为标签时要转为 用 - 连接的形式
            myCom 对应 my-com
    2.Vue.component
        上面的两步可以合并为一步:
        Vue.component('组件名称',{
            template: 'html内容'
        })
        需要注意，模板中的html只能有一个根元素 
    3.将代码写进script标签中
        就像art-template那样，将html模板结构写进到一个script标签中
        ，然后component中的template选项指定为该script标签的id
<script id="tmpl" type="text/html">
    <h1>一个标签来定义的HTML模板结构</h1>
</script>
<div id="app">
    <my-com3></my-com3>
</div>
<script>
    Vue.component('myCom3', {
        template: '#tmpl'
    });
    var vm = new Vue({
        el: "#app",
        data: {
        },
        methods: {
        }
    })
</script>
    4.所谓全局组件，就是说在任意一个Vue控制的html区域中都可以使用

3.使用components定义私有组件
    通过Vue实例化参数 components来定义
    new Vue({
        components: {
            组件名称: {
                template: 'html结构内容'
            }
        }
    })

4.组件中的data
    组件也可以有data属性，但是组件的data必须是个函数，该函数返回一个
    对象

5.组件的data的function必须在方法内部返回一个新的对象，而不能返回外部
的对象，“如果使用外部的对象，那么当实例化多个组件时，使用的将是
    同一个对象中的数据，导致各个组件实例的数据无法独立”，
    以下是一个错误示范:
<div id="app">
    <counter1></counter1>
    <counter1></counter1>
</div>
<script>
    var obj = {
        count:0
    }
    Vue.component('counter1', {
        template: '<button @click="count++" v-text="count"></button>',
        data: function(){
            return obj
        }
    });
    var vm = new Vue({
        el: "#app",
        data: {
        },
        methods: {
        }
    });
</script>

    
6.组件切换
    从一个实例出发，定义2个组件，一个展示登录界面，一个展示注册界面，
    需要做到点击登录和注册按钮就可以在界面中切换这2种组件
    两种方式：
    1.通过v-if v-else决定组件是否展示
        
<a href="#" @click.prevent="flag=true">登录</a>
<a href="#" @click.prevent="flag=false">注册</a>
<login v-if="flag"></login>
<register v-else="flag"></register>
    2.通过component占位符，以及is属性指定要展示的组件
<a href="" @click.prevent="comName='login'">登录</a>
<a href="" @click.prevent="comName='register'">注册</a>
<!--Vue提供了Componentg元素，来展示对应名称的组件-->
<!--component是个占位符,is属性，指定要展示的组件的名称-->
//comName作为变量存于data中
<component :is="comName"></component>

7.组件切换的时候加上动画和mode方式
    用transition将component is:='xx'包裹起来，然后设置过渡类名即可
    注意:
        给transition添加 mode='out-in'，这样可以保证前一个组件的过渡结           
        束再执行第二个组件的进入
