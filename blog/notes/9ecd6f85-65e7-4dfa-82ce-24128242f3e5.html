1.元字符
    .(点)表示的是：除了\n以外的任意一个字符

    []表示范围, [0-9]表示0-9之间的任意一个数字，但是最多表示0-9,也就是说如果想表示100到200之间的的数字字符串不能用[100-200]，这是无效的
        总之[]就是范围内的任意一个字符出现了一次

    | 或者 [0-9]|[a-z] 表示要么是个数字，要么是个小写字母

    ()分组，提升优先级

    下面的也是元字符，也叫限定符，就是限定次数：
        表达式 *: 表示前面的表达式除了0到多次

        表达式 +: 前面的表达式除了1次到多次

        表达式 ?:
            1.前面的表达式出现了0次或1次
            2.组织贪婪模式

        表达式 {} : 更加明确的表示前面的表达式出现的次数
            表达式 {0,}： 表示前面的表示出现了0次到多次
            {4}出现了4次
            {5,10} 出现了5到10次

        ^ 表示以 xx 开始,或者是取非
            ^[0-9] 以0-9开头
            [^0-9] 以非数字开头

        $: 表示以什么结束
            [0-9][a-z]$表示必须以小写字母结束

        严格模式: ^表达式$
            表示整个字符串必须完全符合正则表达式
            ^[0-9][a-z]$: 表示整个字符串必须是 一个数字加一个字母
    转义符:
        \d 数字中任意一个,相当于 [0-9]

        \D 非数字中的任意一个 [^0-9]

        \s 空白符中的一个(空格 tab键 换页等等)

        \S 非空白符

        \w 非特殊符号 等价于 [0-9a-zA-Z_]

        \W 特殊符号

        \b 单词的边界

    练习:
        //        写正则表达式，根据字符串来写正则表达式进行匹配
        //        身份证的正则表达式
        //        1.15位或者18位
                    /(\d{15})|(\d{18})/

        //        座机号码
                /*
                * 010-19876754
                * 0421-2293333
                * */
                /\d{3,4}[-]\d{8}/

                /*
                * qq号码
                * 913852910
                * 从5位到11位
                * */
                /\d{5,11}/
                /*
                * 手机号码
                * 130 131 132 13...
                * */

                /*邮箱的验证
                * sd2113_3.-fd@itcast.com.cn
                * */
        //       -横杠如果没有两边都连接其他字符，那么就表示一个横杠，如果两边都连接了字符，还希望表示为横杠，那么用转义表示 \-
        //        .点如果在中括号中，那么就表示一个点，如果不在中括号中，那么就用转义表示 \.

                /[\da-zA-Z_.-]+[@][\da-zA-Z_.-]+([.][a-zA-Z+]){1,2}/
    创建正则表达式对象:
        1.通过构造函数创建对象
            1.
                var reg = new RegExp(/\d{5}/);
                var str = "我的电话是10086";
                //调用正则表达式的test验证是否匹配，返回是 Boolean,匹配就返回true,否则返回false
                reg.test(str);

            2.可以使用字符串来创建正则表达式,但是需要注意转义
                比如原本正则表达式 /\d{5}/
                放在字符串里面的时候，\需要先转义 "\\d{5}" 这样才和刚才的表达式相符
                因为通过字符串创建出来的正则表达式，是字符串处理完成之后最后显示的结果来表达正则表达式的，
                或者可以这么说，字符串最后显示出来的样子才是真正起作用的正则表达式
                比如,想刚才如果这么来创建正则表达式的话: reg = new RegExp("\d");
                字符串最终显示出来的是 d,那么也就等价于正则表达式 /d/,匹配一个字母 d
        2.通过字面量的方式
            var reg = /正则表达式/

        3.匹配方法
            1.正则对象.test(str)
                返回值是个 true/false
            2.字符串str.match(正则表达式)
                返回值是个数组,数组第一个元素即为匹配成功的子字符串
                返回值结构:
                    {
                        0: "asd"
                        groups: undefined
                        index: 0
                        input: "asdasdasda12312@#123qw"
                        length: 1
                        __proto__: Array(0)
                    }
        4.验证中文字
            [\u4e00-\u9fa5] 就是unicode中 “一“到“龥”的汉字
            在js中啊还可以通过window.escape方法来验证
            window.escape('哈哈')
            window.unescape()
        5.全局模式:
            可以将字符串中所有匹配的子字符串都提取出来
            将所有单个的数字提取出来
                "sda12dlk12xaslk21dkl;q089".match(/\d/g);
        6.配合String.prototype.replace方法来进行字符串替换
            '111111'.replace(/2/, '1')
            但是需要注意，如果正则表达式后面加g全局模式，那么将只会替换第一个匹配的字符
        7.比trim更全面的去除空格的方法
            string.trim可以用来去除字符串前后的空格
            还可以使用正则来去除所有空格
            str.replace(/\s+/g, "");
        8.不区分大小写:
            i后缀,正则表达式后面加上的i后缀表示不区分大小写
            'hHe12hH'.replace(/h/gi, 'aa')将会替换所有的h 和 H
        9.exec方法，这是个正则表达式的方法，效果和在字符串上调用match类似,
            持续调用这个方法，它会依次地的输出每一个匹配的子字符串，一直到不匹配为止，这时返回值为null
