SCSS学习
笔记本：前端学习
1.安装SASS, 目前使用 npm安装, npm i -g sass

2.将 .scss 转为 .css 
    
sass index.scss index.css

3.监听模式
    sass --watch index.scss:index.css

4.scss书写
 变量:
    $变量名来定义变量

 嵌套:
    将层级选择器嵌套书写
        div{
            ul{
                list-style: none
            }
        }

 模块化:
    1.将模块分别 写在 _ 开头的文件中, 比如 _partial.scss
    2.使用@use引入模块
    3.混入mixin
        1.定义混入函数
@mixin transform($property) {
  -webkit-transform: $property;
  -ms-transform: $property;
  transform: $property;
}
        2.使用混入
.box { @include transform(rotate(30deg)); }
