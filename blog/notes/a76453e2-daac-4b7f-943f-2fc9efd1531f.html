4.动画(常用)
    之前利用过渡，可以实现一个效果到另一个效果的转换，但是如果
    元素需要多种效果之间进行变换，就只能动画才能实现了,
    动画的原理:
        定义多个关键帧:
            关键帧1 -> 关键帧2 -> 关键帧3 -> 关键帧4
            每两个关键帧的之间的过渡效果会自动补全，其实也就是多段的
            过渡效果
        定义方式:
            1.@keyframes 动画名称
                定义动画需要用到的帧
                @keyframes moveTest{
                    0% {
                    }
                    50%{
                    }
                    100%{
                    }
                }
            2.目标元素
                animation-name: 动画名称
                animation-duration: 动画耗时
                这两个属性是必须的
            3.animation-timing-function: linear/steps(60) ..
                动画执行时的速度,linear表示匀速的 steps表示按次数/帧数来跳动
            3.animation-iteration-count
                指定动画的播放次数,可以指定固定数值表示指定的次数，
                也可以指定为infinite 表示无穷次
                日常开发中的应用场景: 钟摆的摆动
            4.animation-direction: alternate
                动画交替执行，动画执行结束之后，是从最后状态，还原
                到开始的状态，而不是直接从开发状态从新开始
            5.animation-delay: 2s
                动画延迟2秒后开始
            6.animation-fill-mode:动画结束后的状态
                forwards    保留动画结束时的状态
                backwards   如果动画有初始状态，那么会立即进入到初始状态，即from/0%的状态
                both
            7.animation-play-state: paused running
                控制动画播放的 暂停 播放 状态
5.时钟案例
    border-radius:
        一般写百分比 比 像素要好，因为百分比是相对于元素的实际宽高,
        另外边框圆度指的是，从盒子的角开始往中心偏移指定的距离构造一个
        圆心，然后画1/4个圆，所以指定宽度的一半画1/4个圆的话，那么整个
        盒子将刚好是一个圆； 指定圆心的时候可以x y两个值 100px/50px，加个斜杠，
        这样可以画出椭圆
6.web字体和字体图标:
    1.在线字体:
        去iconfont网站的在线字体功能,输入想要改变字体的内容,下载字体文件，下载完后参照demo文件安装好字体即可
        但是 局限性很明显,生成的字体,仅仅只是我们指定的内容的字会有字体
    2.字体图标:
        以往我们用一些小图标，我们会使用精灵图,但是精灵图使用起来的局限性也很明显
            1.不方便改变，要改的话，得更改整张精灵图
            2.大小尺寸不方便调整
        所以有了字体图标，优点:
            1.所有字体打包成一个文件了
            2.具有矢量性，保证清晰度
            3.改变尺寸很方便，设置字体大小就行了
        使用方式，还是进入iconfont网站在首页下载字体，点击下载素材，然后参照
        demo使用即可，建议使用css方式
7.多列布局
    多列布局的应用场景主要是来源于文字过多的时候，当文章的文字太多时，
    每行的文字都是从头显示到尾，阅读起来很累
    使用:
        column-count:   列数
        column-rule: dashed 每列之间的分割样式,dashed是分割线
        column-gap: 50px 每列之间的间距
        column-width: 100px 每列的宽度,注意这个样式和column-count是冲突的
        浏览器会“取大优先”
8.伸缩布局(重点!)
    1.传统布局的局限性
        以往依赖 display position float属性来给页面进行布局,但是很多
        特殊的布局使用这种传统方式会非常不方便,比如在父容器中,放置
        4个子盒子,并使4个子盒子以相同间距摆放,按照以往的方式,只能
        计算出 具体的像素值 并结合定位或者内外边距来实现,但是一旦用的
        是具体的像素值，那么就很不方便了，比如父容器改变大小，盒子就
        对不齐了；所以传统布局有很多局限性,况且如今很多网页都要求
        有“自适应”能力，尤其是移动端，所以之前传统布局的短板，就更加
        明显了，这个时候就需要h5的伸缩布局
    2.h5的伸缩布局来了
        父容器:
            核心属性
                display: flex 设置父元素内子元素为伸缩布局
                justify-content: xxx 设置子元素布局方式
        代码:
            <style>
                * {
                    margin: 0;
                    padding: 0;
                }
                .box {
                    width: 1000px;
                    height: 200px;
                    border: 1px solid red;
                    box-sizing: border-box;
                    /*设置父容器为盒子，会使每个子元素自动变成伸缩项*/
                    display: flex;
                    /*设置子元素在"主轴"方向上的排列方式*/
                    /*flex-start让子元素从父容器的开始位置开始排列，类似于float left*/
                    /*flex-end让子元素从父容器的结束位置开始排列，类似于float right*/
                    /*center让子元素从父容器的中间位置开始排列*/
                    /*space-between左右对齐父容器的开始和结束，中间平均分布，子元素之间会产生相同的间距*/
                    /*space-round 将多余的空间平均的分布在每个子元素的两边*/
                    justify-content: space-around;
                }
                .first {
                    width: 200px;
                    height: 100%;
                    background-color: red;
                }
                .second {
                    width: 200px;
                    height: 100%;
                    background-color: green;
                }
                .third {
                    width: 200px;
                    height: 100%;
                    background-color: blue;
                }
                .fourth {
                    width: 200px;
                    height: 100%;
                    background-color: pink;
                }
            </style>
            <body>
                <div class="box">
                    <div class="first"></div>
                    <div class="second"></div>
                    <div class="third"></div>
                    <div class="fourth"></div>
                </div>
            </body>
        2.flex-flow 设置父元素内子元素是否换行,以及主轴方向
            上面的布局之后会出现一个现象，即便子元素的宽度之和超过了
            父容器的宽度，子元素也仍然会显示在一行，只不过子元素会进行收缩
            flex-flow其实是flex-wrap和flex-diretion两个属性合计效果
                flex-direction: 子元素的排列方向,设置主轴方向，默认的主轴方向时row(水平方向）
                    还可以设置为column(垂直方向)
                flex-wrap: 当子元素宽度超出时是否换行:
                    wrap换行 nowrap 不换行
            flex-flow就是同时设置这两个属性:
                flex-flow: row wrap
        3.flex属性(设置在子元素上的属性):
            是3个属性的一个综合属性 默认为 0 1 auto后两个属性可省略
                flex-grow flex-shrink flex-basis
                flex-grow: 可以用来扩展子元素的宽度,这个属性应该写在子元素上面
                    设置当前元素应该占用剩余空间的比例值,默认值是0
                flex-shrink 根据设置的值来设置收缩空间,默认值是1,与flex-grow相反，
                    当子元素宽度之和超过父容器时，子元素会收缩，这个值就是用来
                    设置各个子元素收缩的比例的,注意当flex-shrink都设置为0的话，那么
                    每个子元素都不收缩，则内容会溢出
                不过，大多数情况不会单独使用上述2个属性，而是使用flex属性，并且只会指定一个值，即剩余长度
                    flex: value 用来设置子元素占用父容器剩余空间的比例
                    只设置一个值就表示子元素所占用的父容器的宽度的比例,默认值为0
                最多的使用场景：
                    1.父容器设置子元素为伸缩
                    2.子元素不指定宽度，转而设置flex占用比例即可
    3.align-items
        设置子元素在侧轴方向上的对齐方式
        align-items:
            center: 设置在侧轴方向上居中对齐
            flex-start: 顶对齐
            flex-end: 底对齐
