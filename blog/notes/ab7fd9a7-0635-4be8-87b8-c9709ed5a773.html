20191106 Vue生命周期 Vue过渡动画 v-for元素的过渡效果
笔记本：前端学习
1.new Vue之后的生命周期中:
    如果需要通过ajax(vue-resource)加载表格数据,一般初始数据加载放在 
    created()回调当中，因为这个时候data和methods已经
    初始化完成，可以调用了

2.使用vue-resource时全局配置数据接口的域名
    因为每次都要写一个完整的 http://开头的url很麻烦，而且，一旦服务器的域名
    更改了，前端改起来也很麻烦

    也就是我们希望每次请求的都是 this.$http.get('api/getxxx'))这样的url
    
    设置:
Vue.http.options.root = 'http://www.xxx.xxx'
    这样设置后，路径就是从这个配置的路径开始的了，需要注意：
    如果希望url前面加上配置的根路径，那么发送请求时url只能使用相对路径,否则
    不会启用根路径拼接


3.如果你的服务器无法处理 Content-Type: application/json
    那么可以通过emulateJSON: true选项来设置Content-Type:formdata/www-urlencoded
Vue.http.options.emulateJSON = true;
 
4.Vue中的动画
    通过过渡类名可以添加过渡动画,它的原理是会在元素的各个阶段给元素
    添加/删除 指定的类名,然后我们只需要给指定的类名添加动画效果即可
    1.将需要应用过渡元素使用 transition包裹起来
        给元素应用v-if属性，使其按照条件显示与隐藏
    2.定义样式
/*自定义两组样式来控制transition内部的元素实现动画*/
/*进入之前元素的起始状态*/
.v-enter,
.v-leave-to{
    opacity: 0;
    transform: translateX(10px);
}
/*进入/离场 时候的时间段*/
.v-enter-active,
.v-leave-active{
    transition: all .4s ease;
}

5.修改v-前缀
    给transition一个name属性,比如 <transition name='my'>
    然后再定义样式，这个时候样式的类名由v-改为 my-
6.使用第三方animate.css类库实现动画
    1.引入 animate.css
    2.
<input type="button" value="toggle" @click="flag=!flag">
<!--使用transition元素，把需要被动画控制的元素包裹起来-->
<!--transition元素是vue官方提供的-->
<!-- duration表示动画耗时 -->
<transition enter-active-class="animated bounceIn" leave-active-class="animated bounceOut" :duration="1000">
    <h3 v-if="flag">这个h3</h3>
</transition>

7.钩子函数实现半场动画
    Vue提供了一些元素渲染过程中特殊的事件
    before-enter enter after-enter enter-cancelled before-leave leave after-leave leave-cancelled    
    通过v-on监听这些事件，然后绑定事件处理函数即可处理各个阶段的动画
    如果是半场动画，则只要定义enter部分就可以了
    
    具体做法:
    1.使用 v-on: 或者 @ 绑定钩子函数
<transition @before-enter="beforeEnter" @enter="enter" @after-enter="afterEnter">
    <div id="ball" v-show="flag"></div>
</transition>
    2.在methods中定义对应的钩子函数
    var vm = new Vue({
        el: "#app",
        data: {
            flag: false
        },
        methods: {
//            钩子函数的第一个参数 el，表示要执行动画的那个DOM元素,是个原生的JS DOM对象
            beforeEnter(el){
                console.log(1);
            },
            enter(el){
                console.log(2);
            },
            afterEnter(el){
                console.log(3);
            }
        }
    })
    3.必须在enter中调用el.offsetWidth，动画才会顺利执行，老师他自己也
      不知道为什么

8.使用transition-group实现列表动画
        如果需要渲染的元素是通过v-for循环渲染出来的， 不能直接使用
    transition包裹，而是使用transition-group来包裹
        注意，如果要使 v-for循环 创建的元素设置动画，必须为每个元素
        设置:key属性

9.transition-group的 appear 和 tag属性的作用
    1.appear可用于辅助动画效果，只要给transition-group添加了这个属性，
    就会在页面刚展示出来的，有一个渐变的入场效果
    2.默认情况下，Vue会使用一个span元素将循环渲染出的所有元素包裹起来
        这不符合w3c规范,而tag属性可以指定将来用于包裹列表的元素
        <transition-group tag='ul>
            <li v-for='item in list' :key='item.id'></li>
        </transition-group>
        这样，渲染的元素会用ul包裹起来
