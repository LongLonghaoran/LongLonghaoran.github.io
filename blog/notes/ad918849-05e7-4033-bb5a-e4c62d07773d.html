3.数组 冒泡排序
    1.数组的定义
        1.通过构造函数创建数组
            var 变量名 = new Array();
            创建了一个空数组，没有数据
            如果有一个参数则表示长度，如果有多个参数，则表示数组元素
            var arr = new Array(5); // [undefined, undefined, undefined, undefined,undefined]
            var arr2 = new Array(1,2,3,4,5)
        2.通过字面量的方式创建数组
            var arr = [1,2,3]

        3.数组的反转,所谓数组的反转指的是,更改一个数组中的元素的排列顺序
            for(i=0; i<arr.length/2;i++){
                arr[i]=arr[i]^arr[arr.length-i-1]
                arr[arr.length-i-1]=arr[i]^arr[arr.length-i-1]
                arr[i]=arr[arr.length-i-1]^arr[i]
            }
        4.冒泡排序,最简单的排序
            一个嵌套循环,每次都从第一个元素开始，让它和它之后的一个元素相比较，
            并将较大/较小的值放在后面,这样的循环一次之后就将最大/最小的值放到
            最后面，之后的每次循环都继续从第一个开始，循环到第（总数-1）个元素，
            再将剩余的元素的最大/最小放到最后
            for(var i=0; i<arr.length; i++){
                var temp = 0;
                for(var j=0; j<arr.length-i; j++){
                    if(arr[j]>arr[j+1]){
                        temp = arr[j];
                        arr[j] = arr[j+1];
                        arr[j+1] = temp;
                    }
                }
            }
