笔记本：前端学习
1.复习
    offset 系列
        offsetLeft offsetTop offsetWidth offsetHeight
            offsetWidth 宽度(不含边框)
            offsetLeft: 左边的边距 left
    scroll 系列
        scrollWidth 宽度(含边框)
        scrollLeft: 左边滚动的距离
    client 系列
        element.clientWidth 可视区域的宽度（不含边框）
        element.clientLeft: 左边框的宽度(其实并非左边距，而是左边边框的宽度)
        event.clientX 可视区域的横坐标
        event.clientY 可视区域的纵标坐

1.案例:图片跟着鼠标飞:
    在之前图片跟着鼠标飞的核心要点是获取当前鼠标距离 页面的左边和右边的距离
    clientX 获取的是鼠标在当前可视区域的横坐标
    clientY 获取的是鼠标在当前可视区域的纵坐标
    有个办法是 clientX + scrollLeft 即为真正的距离页面左边的距离

    还有一个解决方案:
        1.如果代码是在事件处理函数中，那么通过事件对象是可以直接获取到
        距离页面的距离的,通过 event.pageX event.pageY即可直接获得,
        即: event.pageX = event.clientX + getScroll().left
        其中获取当前页面滚动值也需要兼容代码:
            function getScroll() {
              return {
                left:window.pageXOffset||document.body.scrollLeft||document.documentElement.scrollLeft||0,
                top:window.pageYOffset||document.body.scrollTop||document.documentElement.scrollTop||0
              }
            }

    兼容代码:
        var evt = {
            //获取事件参数对象e
            getEvent: function(e) {
                return window.event || e;
            },
            //获取可视区域的横坐标
            getClientX: function(evt){
                return this.getEvent(evt).clientX;
            },
            //获取可视区域的纵坐标
            getClientY: function(evt){
                return this.getEvent(evt).clientY;
            },
            获取向左卷曲出去的横坐标:
            getScrollLeft: function(){
                return window.pageXOffset || document.body.scrollLeft || document.documentElement.scrollLeft || 0;
            },
            获取向上卷曲出去的纵坐标:
            getScrollTop: function(){
                return window.pageXOffset || document.body.scrollLeft || document.documentElement.scrollLeft || 0;
            },
            获取元素相对于完整页面的横坐标:
            getPageX: function(evt){
                return this.getEvent(evt).pageX ? this.getEvent(evt).pageX : this.getClientX(evt) + this.getScrollLeft(evt);
            },
            获取元素相对于完整页面的纵坐标:
            getPageY: function(evt){
                return this.getEvent(evt).pageY ? this.getEvent(evt).pageY : this.getClientY(evt) + this.getScrollTop(evt);
            }
        }

5.自定义滚动条
    要求:
        1.滚动高度自动计算
        2.文本不会因为滚动而被选中
    思路:
        1.在滚动条上注册点击事件，当鼠标点击时，就记录鼠标距离页面顶部的纵坐标值
            并根据纵坐标值以及滚动条的纵坐标值，与之相减，来获取点击位置与滚动条
            顶部之间的距离
        2.在点击事件中继续注册鼠标移动事件mousemove,在移动事件中，获取鼠标的纵坐标值，
            将该值再减去刚才算出来的与滚动条顶部的距离值，即为在纵向上需要移动的距离
        3.文本内容需要向下移动，移动的距离，根据比例来算出，这个比例应该是：
            文本移动的距离 / 文本可以移动的最大距离=滚动条移动的距离 / 滚动条可以移动的最大距离
            即：
                文本移动的距离=文本可以移动的最大距离 * 滚动条移动的距离 / 滚动条可以移动的最大距离
            再而得出:
                文本移动的距离= (文本盒子的高度-父盒子的高度) * 滚动条移动的距离 / (滚动条父盒子的高度-滚动条的高度)
