京东项目除了商品列表和侧边电梯导航以外 结束

展示一个没有两侧边框的表格:
    
[图片/附件]

这个表格用ul>li实现,我们可以用一个父级div包裹一个ul>li，然后给父盒子div一个上边框，并且给li下右边框以及浮动,
div宽度190px，li宽度47px, 边框1px solid #ccc,这样会有2个问题:
    1.4个li在一排的宽度已经达到191px,已经超过了div,所以一排只会显示3个，第4个则被挤下去
    2.最右边以及最下边是不需要边框的,但是现在显示了边框
结果会是这样的:
[图片/附件]
要实现目标效果的方法就是:
    我们给li的父盒子ul元素，一个宽高,这个宽高大于div,这样不旦可以容纳进4个li一排，而且设置了div的overflow:hidden之后，超出部分还会被隐藏，
    恰好挡住了不需要的边框
    代码如下:
        html:
<div class="jd-service">
                    <ul>
                        <li>
                            <a href="#">
                                <i></i>
                                <p>机票</p>
                            </a>
                        </li>
                        ...之后11个li
                    </ul>
                </div>

css:
    
* {
    margin: 0;
    padding: 0;
}
li {
    list-style: none;
}
.jd-service {
    width: 190p;
    height: 209px;
    overflow: hidden;
}
.jd-service ul {
    width: 192px;
}
.jd-service li {
    width: 47px;
    height: 69px;
    float: left;
    border-right: 1px solid #ccc;
    border-bottom: 1px solid #ccc;
}
