Data
主要有Dataset和Dataloader两个类，Dataset用于存储样本和对应的标签，Dataloader用于封装遍历Dataset的逻辑，torch自带很多数据集
同时有一些领域特定的库，可以分别提供不同领域的数据集
比如torchvision torchaudio torchtext，各自有各自的Dataset分别用于提供
领域特定的数据集
Dataset有个transform参数用于转换数据格式（转换为某种矩阵）
领域特定的这些库本身也提供了一些转换方法：
import torch
from torch import nn
from torch.utils.data import DataLoader
from torchvision import datasets
from torchvision.transforms import ToTensor
# Download training data from open datasets.
training_data = datasets.FashionMNIST(
    root="data",
    train=True,
    download=True,
    transform=ToTensor(),
)

# Download test data from open datasets.
test_data = datasets.FashionMNIST(
    root="data",
    train=False,
    download=True,
    transform=ToTensor(),
)
