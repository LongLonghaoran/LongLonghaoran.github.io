1.scroll系列:
    同offset类似,有 scrollWidth scrollHeifht scrollLeft scrollTop四个常用属性
    scrollHeight,元素实际的高度（不含边框)  而offsetHeight(含边框)是元素css的高度
    但是scrollHeight是元素在页面中实际展示出来的高度，也就是说如果一个div设置了
    width height的情况下，offsetWidth offsetHeight获取的就是元素宽高加上边框的值，
    而scrollHeight scrollWidth会是经过计算的实际宽高
    scrollTop表示向上滚动的距离
    scrollLeft向左滚动的距离

    div的滚动事件:
        div.onscroll
    获取整个页面滚动距离的兼容代码:
        同理我们可以获取整个浏览页面的滚动距离，但是由于各个浏览器支持情况不统一，所以
        需要写兼容代码:
            function getScroll(){
                // document.documentELement获取的是html元素 document.body是body元素,如果都获取不到就给个0
                return {
                    top: window.pageYoffset || document.documentElement.scrollTop || document.body.scrollTop || 0,
                    left: window.pageXoffset || document.documentElement.scrollLeft || document.body.scrollLeft || 0
                };
            }

    function getStyle(element, attr){
        return window.getComputedStyle ? window.getComputedStyle(element,null)[attr] : element.currentStyle[attr]
    }

7.client系列:
    回顾:
        之前讲了 offset(获取宽高 left top parent) scroll(获取画面/元素滚动距离)系列
        offsetParent:
            HTMLElement.offsetParent 是一个只读属性，返回一个指向最近的（closest，指包含层级上的最近）包含该元素的定位元素。
            如果没有定位的元素，则 offsetParent 为最近的 table, table cell 或根元素。
            当元素的 style.display 设置为 "none" 时，offsetParent 返回 null。
            offsetParent 很有用，因为 offsetTop 和 offsetLeft 都是相对于其内边距边界的。
        offsetWidth 宽度(不含边框)
        scrollWidth 宽度(含边框)
        clientWidth 可视区域的宽(不含边框)
        offsetLeft: 左边的边距 left
        scrollLeft: 左边滚动的距离
        element.clientWidth 可视区域的宽度（不含边框）
        element.clientLeft: 左边框的宽度(其实并非左边距，而是左边边框的宽度)
        event.clientX 可视区域的横坐标
        event.clientY 可视区域的纵坐标
        element.pageY: 当前可视区域相对于页面顶部的坐标
        element.pageX: 当前可视区域相对于页面左边的坐标
