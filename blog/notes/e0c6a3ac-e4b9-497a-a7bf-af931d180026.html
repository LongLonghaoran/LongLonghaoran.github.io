1.canvas中的圆是用弧度来表示的
    复习一下弧度:
        弧长等于半径的一个圆弧就是一弧度的圆，而假设半径为r
        那么圆周长为 2 * Pi * r, 那么一个圆的弧度数就等于 2 * pi * r / r = 2 * pi
        即: 一个圆的弧度为 2 Pi
    canvas绘制曲线:
        1.利用 “以点连线”的原理来绘制曲线，在昨天的内容已经写过“渐变”
        可以体会到一条线，它可以是很多个点 或者说多段 1像素的线段 连接起来从而成为一条线的。
        2.利用数学公式来计算出绘制一条线所需所有点，并且遍历它来将全部绘制出来
            比如，一条直线可以用一个 二元一次方程来绘制:
            y = 2x+1;
                for(var i = 1; i< 600; i++){
                    var x = i;
                    var y = 2*x + 1;
                    ctx.lineTo(x, y);
                }
                ctx.stroke();
            那么以此类推，我们可以将公式换成2元2次方程，那么绘制出来的应该就是曲线了（抛物线）
            可以使用Math.pow(x, n)来计算x的n次方
            或者，更明显的，我们直接使用 正玄函数来绘制正玄曲线
                var canvas = document.querySelector("canvas");
                var ctx = canvas.getContext("2d");
                for(var i = 1; i< 600; i++){
                    var x = i;
                    var y = Math.sin(x*2*Math.PI/600)*100 + 100;
                    ctx.lineTo(x, y);
                }
                ctx.stroke();
    canvas中绘制圆弧:
        1.直接使用canvas的api绘制圆弧:
            1.确定圆心坐标
            2.确定半径
            3.确定弧度
            使用ctx.arc(圆心x坐标, 圆心y坐标, 半径长度(不带单位,直接写数字), 开始角度(用弧度表示), 结束角度(用弧度表示), 是否为顺时针);
                    var w = canvas.width;
                    var h = canvas.height;
            //        默认是顺时针，从水平线的正方向开始为0
            //        ctx.arc(w/2, h/2, 150, Math.PI/2, Math.PI);
            //        指定为逆时针
                    ctx.arc(w/2, h/2, 150, Math.PI/2, Math.PI, true);
                    ctx.stroke();
            角度的变换方向是，顺时针变化为角度/弧度变换的正方向
