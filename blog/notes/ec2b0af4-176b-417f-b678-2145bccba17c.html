笔记本：前端学习
Jquery
1.操作样式
    css方法
        1.修改单个样式:
            $().css("backgroundColor": "pink")
        2.修改多个样式:
            1.链式调用:
                css().css().css()....
            2.传个对象
                css({
                    backgroundColor: 'pink'
                })
        3.获取元素的样式
            在js中通过style获取样式只能获取行内样式，如果要获取style属性中定义的样式
            的话只能通过offset window.getComputedStyle这样的属性去获得

            Jquery中只要通过css(attr)就可以直接获得元素的样式属性值，自动的处理好了兼容性问题
            它会获得第一个元素的样式属性值
    class操作:
        1. $().addClass()
            在元素之前已有类的基础上继续增加类
        2. $().removeClass()
            移除元素中其中一个类
        3. $().hasClass(className) 是否包含某个类
        4. $()选择出来的元素，不管是单个元素还是多个元素，在Jquery中包含一个整个的集合中
            我们既可以同时对这些元素操作 $().css(),也可以取出一个单独的元素操作 $().eq(0).css,
            操作的方式都是一样的
        5.toggleClass(className) 切换类名,有则取出，无则添加
    属性操作:
        1.$().attr(属性名, 属性值)  移除属性 removeAttr(属性名)
        2.和css方法类似，可以更改属性，但是不同于css, attr它可以设置不属于标准属性的自定义
        3.一次更改多个属性:
            attr({
                title: '',
                alt: 'x',
                aa: '123' //自定义属性
            })
        4.属性操作的特例
            checkbox的无法通过attr("checked", 1)来选中
            有2种解决方法：
                1.转为dom对象然后通过checked赋值,
                    $("[type='checkbox']")[0].checked = true;
                    $("[type='checkbox']")[0].checked = false;
                2.通过prop方法
                    $().prop("checked", true);
            出现这种情况的原因:
                对于没设置的属性 比如checked selected disabled，它获取的是undefined
                对于Boolean类型的属性，不要用attr方法
        5.不论是prop还是attr方法，他们获取的总是jq对象中的第一个dom对象的属性值
    Jquery动画:
        1.三组基本动画
            显示隐藏,原理就是控制元素的display属性为 none 或者 block:
                show(毫秒数, fn) 显示
                hide(毫秒数, fn) 隐藏
            滑入滑出,通过height margin-top实现
                slideDown(毫秒数, fn) 滑入
                slideUp(毫秒数, fn) 滑出
                slideToggle(毫秒数, fn) 在滑入和滑出两种状态中切换
            淡入淡出
                fadeIn fadeOut fadeToggle
                用法同上
        2.动画函数中的fn都是回调函数，当动画执行完成之后就会执行回调函数

        3.自定义动画 animate方法，意味之前手写的js版的animate方法，完全用不到了
            $().animate( prop, speed, easing, callback )
                prop 一个由属性名: 属性值 组成的对象
                speed 速度/花费时间/毫秒数
                easing 速度变化趋势: 默认 swing摇摆速度（由慢变快再变慢） linear匀速
                callback: 回调函数，执行完成之后就执行
        4.Jquery动画的队列：
            只要触发了的动画就不会停止，比如在手风琴案例中，鼠标一次性滑过整个ul，
            即便鼠标已经离开了ul范围，但是动画仍然会挨个执行结束，但是我们预期的应该是
            鼠标离开整个大盒子，所有动画就都停止，并且所有元素都复原
            我们需要每次执行一个动画的时候，停止上一个动画
            $().stop(),停止该Jquery对象正在执行的动画，让它立即变换到目标位置
            例子：
                $("li").each(function(index, ele){
                    ele.style.backgroundImage = 'url(images/'+(index+1)+'.jpg)'
                });
                $("li").on("mouseenter", function(){
                    $(this).stop().animate({width:'600px'}).siblings().stop().animate({width: '100px'})
                });
            stop两个参数:
                1.clearQueue,是否清除动画队列
                    一个元素有可能有多个动画依次执行，默认情况stop只会停止正在执行的动画
                    clearQueue为true,表示后续所有动画都不执行
                2.jumpToEnd:是否直接跳转到最终效果，默认情况下，stop后，元素就保留动画停留时
                    的样式。如果为true,就直接变化到最终的样式
    Jquery创建节点 添加节点:
        创建:
            $(html字符串)
        添加:
            介绍6个基本方法，就够用了：
                1.添加子元素，在最后添加
                    $().append(html字符串|$())
                    参数可以是html字符串或者jq对象
                2.$().appendTo($())，将元素添加到另一个元素的结尾处
                3.$().prepend($()) 在最前面添加子元素
                4.$().prependTo($())
                5.$().after($()) 在自身后面添加作为兄弟元素
                6.$().before($()) 在自身前面添加作为兄弟元素
        清空:
            1.$().html(""); 等价于 innerHTML = ""
                但是该方法会导致内存泄漏,因为它的原理仅仅是将内容清空，而不会内部的事件给清空，
                比如:
                    <div>
                        <p></p>
                    </div>
                    $("div").html("")
                    如果p上面有事件的话，那么清空div的内容，是不会清空p上面的事件的
            2.$().empty()
                不仅将html内容清空,而且将相关的事件都清空，阻止内存泄漏
        删除:
            $().remove();
        克隆:
            $().clone()
            功能:
                克隆匹配的DOM元素并且选中这些克隆的副本。
                在想把DOM文档中元素的副本添加到其他位置时这个函数非常有用。
            两个参数:
                1:一个布尔值（true 或者 false）指示事件处理函数是否会被复制。

                2:一个布尔值，指示是否对事件处理程序和克隆的元素的所有子元素的数据应该被复制。
