node.js是什么
1.是一个javascript运行时环境，可以脱离浏览器执行，但是node.js中没有dom bom对象,其中只有    ECMAscript,也就是基本的语法
     事件驱动 非阻塞IO模型

2.浏览器引擎
    Apple Safari的nitro
    Chrome的V8（目前公认最快的js引擎）

3.基本使用
    编写一个xx.js文件
    命令行使用node命令执行 node xx.js

4.读取文件:
    引入fs模块即可，在nodejs中需要使用其他的api前都必须先require引入
    fs = require('fs')
    使用readFile来读取文件: readFile(文件路径, 回调函数)
    readFile("xx", function(error, data){})
    其中error是错误对象, data是读取到的数据

    读取到的数据会转为16进制显示，可以通过toString()方法将其转为可读的形式
var fs = require('fs')
fs.readFile('../README.md', function(error, data){
    // 如果读取失败,error就是错误对象, data 就是 null
    // 读取成功,error就是null, data就是读到的数据
    console.log(data.toString());
})

5.写文件
fs.writeFile(路径, 内容, 回调函数(error))
fs = require('fs');
// 只有一个参数，写入成功则error为null，写成失败则error就是错误对象
fs.writeFile('./你好.txt', '你好啊', function (error) {
    console.log(error);
})
