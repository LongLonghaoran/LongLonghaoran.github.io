通过Linux发送邮件
笔记本：Linux
1.安装邮件客户端mailx: yum install -y mailx
2.安装邮件服务器sendmail: yum install -y sendmail
3.安装procmail: yum install -y procmail
4.给邮件服务器/etc/alias目录增加进入权限:chmod +x /etc/aliases
5.启动sendmail服务器: systemctl start sendmail
6.发送邮件 mail -s "test" 18621781488@163.com,之后进入邮件正文输入，输入完成后按下.再按回车即可

参考:
https://cloud.tencent.com/developer/article/1725000

故障排查:
可以通过journalctl -xe -u sendmail查看sendmail的错误日志
    我在华为云服务器上测试发送邮件失败，报错显示connection timeout，猜测是被华为云网关拦截了
    通过telnet 目标邮件服务器主机名 25 测试，发现没有返回220开头的问候响应，说明是练不通的，证实了我的想法，之后在其他服务器上测试发送邮件，是可以的了
