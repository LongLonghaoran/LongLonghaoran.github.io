            7.格式化日期
                function getDate(dt) {
                    //获取年
                    var year = dt.getFullYear();
                    //获取月
                    var month = dt.getMonth() + 1;
                    //获取日
                    var day = dt.getDate();
                    //获取小时
                    var hour = dt.getHours();
                    //获取分钟
                    var minute = dt.getMinutes();
                    //获取秒
                    var second = dt.getSeconds();
                    month = month < 10 ? "0" + month : month;
                    day = day < 10 ? "0" + day : day;
                    hour = hour < 10 ? "0" + hour : hour;
                    minute = minute < 10 ? "0" + minute : minute;
                    second = second < 10 ? "0" + second : second;
                    return year + "年" + month + "月" + day + "日 " + hour + ":" + minute + ":" + second;
                }
    3.String字符串对象:
        1.一个特性,js中没有单独的字符类型，只有字符串，字符串实际上是一个字符数组
        2.字符串的不可变性
            当我们新建了一个字符串对象之后，虽然可以通过索引方式访问字符串的每一个字符，
            但是我们无法去改变它 var str = '123'; str[1] = 'bbb'这样是改变不了的
            ，我们只能通过重新给变量赋予一个新的字符串对象来更改字符串 str = 'abc'
        3.字符串的方法
            0.以下为字符串常用的方法，当然需要注意，字符串对象的不可变性，所以不管使用
                哪个方法，字符串原本的值都是不会改变的，方法都是会返回一个新的值，需要
                使用另一个变量来接收
            1.charAt(index),根据一个索引值来获取指定的字符,不给参数默认为0,索引从0开始,
            和数组一样，第一个元素的索引是0
            2.String.fromCharCode(ascii)将ascii值转为对应的字符,注意这是个静态方法
            3.concat('a','b','c') 字符串拼接并返回一个新的方法
            4.replace(s1, s2) 字符串替换,将s1替换为s2,注意s1可以是一个字符串或者一个正则表达式
            5.slice(index1, index2)根据给出的2个索引截取字符串
                str.slice(beginIndex[, endIndex])
                参数节
                beginIndex
                从该索引（以 0 为基数）处开始提取原字符串中的字符。如果值为负数，会被当做 strLength + beginIndex 看待，这里的strLength 是字符串的长度（例如， 如果 beginIndex 是 -3 则看作是：strLength - 3）
                endIndex
                可选。在该索引（以 0 为基数）处结束提取字符串。如果省略该参数，slice() 会一直提取到字符串末尾。如果该参数为负数，则被看作是 strLength + endIndex，这里的 strLength 就是字符串的长度(例如，如果 endIndex 是 -3，则是, strLength - 3)。
            6.indexOf(str) 查找一个字符串的索引值
            7.split(str)使用指定的分隔符,将字符串分割为多个子字符串,返回一个新的数组
            8.substr(start, length)返回从指定索引位置开始截取length个数的字符串
            9.案例 找出字符串中所有的 o 出现的位置

        8.Array.from() 方法从一个类似数组或可迭代对象中创建一个新的，浅拷贝的数组实例。
            它可以从一个类数组来创建一个真数组，比如:
                var obj = {0: 'a', 2: 'b', length: 2}
                Array.from(obj)
            它还有额外的两个参数:
                Array.from(arrayLike[, mapFn[, thisArg]])
                参数节
                arrayLike
                想要转换成数组的伪数组对象或可迭代对象。
                mapFn (可选参数)
                    如果指定了该参数，新数组中的每个元素会执行该回调函数。
                    比如:
                        Array.from({0:1,1:2, length: 2}, x=>x+1)
                        第一个参数是个伪数组（伪数组还有很多其他的形式，总之就是它并不是Array实例，但是可以按照Array方式来使用）
                        第二个参数是个箭头函数，x表示数组中的每一个元素
                thisArg (可选参数)
                可选参数，执行回调函数 mapFn 时 this 对象。
                返回值节
                一个新的数组实例

            由于它取决于具体实现，因此无法保证排序的时间和空间复杂性。
            使用方法:
                arr.sort([compareFunction])
                参数节
                compareFunction 可选
                用来指定按某种顺序进行排列的函数。如果省略，元素按照转换为的字符串的各个字符的Unicode位点进行排序。

3.基本包装类型
    在上面讲了内置对象有很多的方法和属性,但是实际开发中也会经常直接使用字面量来创建
    var str = '123'; var num =123;
    这些变量也可以调用对应对象中的方法,比如
    str.split('1')
    num.toString()
    这是由于，在调用属性或者方法的时候，变量已经转换为”基本包装类型“了
    常见的也就这3种情况,number boolean string3种基本类型，对应着3种对象类型Number Boolean String
    (总共有6种数据类型 number string boolean null Object undefined)
    一个特殊的情况:
        var flag = new Boolean(false);
        var result = flag&&true;
        结果是 true, 由于 flag是一个包装类型了，所以逻辑表达式第一部分，其实并非false,而是个对象，对象存在即为true
