    3.一个最基本的例子，在canvas画布中绘制一条直线
        <body>
            <canvas width="600" height="400"></canvas>
            <script>
                var myCanvas = document.getElementsByTagName("canvas");
        //        获取上下文,专业术语叫做获取上下文(暂时不太明白为什么要这么叫)
                var ctx = myCanvas[0].getContext('2d');
        //        画一条直线
        //        移动画笔,移动至在canvas中的横纵坐标分别为 100 100的位置
                ctx.moveTo(100, 100);
        //        绘制轨迹，或者说叫做路径
                ctx.lineTo(200, 100);
        //        给轨迹描边
                ctx.stroke();
            </script>
        </body>
        解析：
        1.为什么画布尺寸需要通过canvas的width height属性来指定，如果通过style设置呢?
            画布大小跟元素是分别独立存在的
            通过style更改的只是元素的大小，并不是画布的大小,如果
            只通过css来改变画布的尺寸的话，那么里面画的图片也会按照比例
            进行缩放
        2.画布中的线条是灰色的且线条宽度是2px:
            默认颜色是黑色，默认宽度是1px
            原因是 因为原本线条是在像素分割线的上下两边各占了0.5px，且颜色是黑色的
            但是由于浏览器无法展示0.5px，所以只能解析成各占1px，总共占用了2px了，然后
            将颜色减淡，于是就有了2px宽度的灰色线条，
            也就是说，上面的例子中，如果我们设置moveTo(100, 100.5) lineTo(200, 100.5)
            那么线条就会显示成1px了

    6.canvas中的渐变
        它的原理是: 通过将一条线的每个像素格子赋予逐渐变化的不同的颜色来实现的
        <canvas width="500" height="500"></canvas>
        <script>
            var canvas = document.querySelector("canvas");
            var ctx =canvas.getContext('2d');
            ctx.lineWidth = 50;
            for(var i = 0; i<=255;i++){
                ctx.beginPath();
                ctx.moveTo(100+i, 100);
                ctx.lineTo(101+i, 100);
                ctx.strokeStyle = `rgb(${i},${i},${i})`;
                ctx.stroke();
            }
        </script>
